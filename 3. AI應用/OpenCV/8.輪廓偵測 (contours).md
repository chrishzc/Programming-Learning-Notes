# OpenCV 輪廓偵測與幾何特徵

> [!ABSTRACT]
> 本篇筆記介紹如何偵測並繪製影像中的輪廓（Contours），深入說明輪廓與邊緣的差異、樹狀階層結構（Hierarchy）、邊界近似方法，以及如何計算物體的外接幾何形狀（正外接矩形、最小外接矩形、最小外接圓）與面積/周長。

---

## 1. 輪廓與邊緣的差異

- **邊緣 (Edges)**：影像中像素亮度劇烈變化的點組成的集合，線條不一定是封閉的（例如：一條線段或孤立的弧）。
- **輪廓 (Contours)**：由邊緣點相連所組成的**閉合曲線**。輪廓能夠代表一個完整的物體外形，具有清晰的拓撲結構與幾何特徵。

---

## 2. 尋找與繪製輪廓

### 2.1 尋找輪廓 (`cv2.findContours`)
在尋找輪廓前，必須先將影像轉為**灰階**並進行**二值化（Binarization）**（使用 `cv2.threshold` 或 `cv2.Canny`）。
> [!IMPORTANT]
> OpenCV 會將**白色像素 (255)** 視為前景（目標物體），**黑色像素 (0)** 視為背景。

#### 語法：
```python
contours, hierarchy = cv2.findContours(binary_img, mode, method)
```
- **`contours`**：儲存所有檢測到的輪廓的 Python List。List 中的每個元素都是一個形如 `(N, 1, 2)` 的 NumPy 陣列，代表一個輪廓上的點座標。
- **`hierarchy`**：輸出階層陣列，形狀為 `(1, N, 4)`，儲存每個輪廓的嵌套包含關係。

#### 常用輪廓檢索模式 (Mode)
| 檢索模式 | 說明 |
| :--- | :--- |
| `cv2.RETR_EXTERNAL` | **最常用**。只偵測最外層的輪廓，忽略內部嵌套的所有子輪廓。 |
| `cv2.RETR_LIST` | 偵測所有輪廓，但**不建立**任何層次嵌套關係（全部扁平化為同級別）。 |
| `cv2.RETR_CCOMP` | 偵測所有輪廓，建立兩層階層關係：外邊界與內孔邊界。 |
| `cv2.RETR_TREE` | 偵測所有輪廓，並建立**完整的樹狀嵌套關係**。 |

#### 常用輪廓近似方法 (Method)
- **`cv2.CHAIN_APPROX_NONE`**：儲存輪廓上的所有邊界點（佔用記憶體多）。
- **`cv2.CHAIN_APPROX_SIMPLE`（推薦）**：壓縮水平、垂直與對角方向的線段，**只保留頂點座標**（例如矩形只存 4 個角點，大幅節省儲存與運算開銷）。

---

### 2.2 繪製輪廓 (`cv2.drawContours`)
尋找到輪廓後，可以使用 `cv2.drawContours` 將其繪製回彩色原圖上。

#### 語法：
```python
cv2.drawContours(image, contours, contourIdx, color, thickness)
```
- `image`：要在上面畫圖的目標影像（必須是 BGR 彩色圖）。
- `contours`：傳入 `findContours` 取得的輪廓 List。
- `contourIdx`：要畫第幾個輪廓。**設為 `-1` 代表繪製所有輪廓**。
- `color`：繪製的 BGR 顏色。
- `thickness`：線條粗細。**若設為 `-1`（或 `cv2.FILLED`），則會將輪廓內部填滿顏色**。

---

## 3. 輪廓嵌套關係（階層結構 Hierarchy）

`cv2.findContours` 回傳的 `hierarchy` 是一個維度為 `(1, N, 4)` 的 NumPy 陣列。
對於第 `i` 個輪廓 `contours[i]`，它的包含關係表示為：
$$\text{hierarchy}[0][i] = [\text{Next}, \text{Previous}, \text{First\_Child}, \text{Parent}]$$

- **`Next`**：同階層的下一個輪廓索引。
- **`Previous`**：同階層的前一個輪廓索引。
- **`First_Child`**：其包含的第一個子輪廓（內層）索引。
- **`Parent`**：包含它的父輪廓（外層）索引。
- *提示：若無對應關係，該數值為 `-1`。*

## 4. 幾何特徵計算

在輪廓提取後，通常需要計算輪廓的面積與周長來過濾雜訊，或者獲取其外接幾何形狀。

### 4.1 面積與周長
```python
# 1. 計算輪廓面積 (像素數目)
area = cv2.contourArea(cnt)

# 2. 計算輪廓周長 (閉合曲線長度)
# closed=True 代表為閉合曲線
perimeter = cv2.arcLength(cnt, closed=True)
```

---

### 4.2 正外接矩形 (`cv2.boundingRect`)
計算能完全包圍輪廓且**邊與圖像座標軸平行**的最小正矩形。

#### 程式範例：
```python
# 傳入單一輪廓 cnt，返回左上角座標與寬高
x, y, w, h = cv2.boundingRect(cnt)

# 在影像上繪製綠色矩形
cv2.rectangle(image, (x, y), (x + w, y + h), (0, 255, 0), 2)
```

---

### 4.3 最小外接矩形 (`cv2.minAreaRect`)
計算能包圍物體的最小矩形，**可順應物體角度旋轉**（斜矩形）。

> [!CAUTION]
> **如何繪製旋轉矩形？**
> `cv2.minAreaRect` 返回的是含有中心與角度的結構，不能直接傳入 `cv2.rectangle`。
> 必須先使用 **`cv2.boxPoints`** 取得矩形的 4 個頂點座標，將其轉為整數後，使用 **`cv2.drawContours`** 繪製。

#### 程式範例：
```python
import numpy as np

# 計算最小外接矩形
# 返回結構為: ((中心x, 中心y), (寬, 高), 旋轉角度)
rect = cv2.minAreaRect(cnt)

# 取得矩形的 4 個角點座標
box = cv2.boxPoints(rect)
box = np.int0(box)  # 轉換為整數座標

# 繪製紅色旋轉外接矩形 (把 box 當作單一輪廓繪製)
cv2.drawContours(image, [box], 0, (0, 0, 255), 2)
```

---

### 4.4 最小外接圓 (`cv2.minEnclosingCircle`)
計算能包圍物體的最小外接圓。

#### 程式範例：
```python
# 返回圓心座標與半徑
(cx, cy), radius = cv2.minEnclosingCircle(cnt)
center = (int(cx), int(cy))
r = int(radius)

# 繪製藍色外接圓
cv2.circle(image, center, r, (255, 0, 0), 2)
```

---

## 5. 輪廓檢測完整實作範例

```python
import cv2
import numpy as np

# 1. 建立測試彩色畫布與一個斜置三角形
img = np.zeros((400, 400, 3), dtype=np.uint8)
pts = np.array([[100, 150], [250, 100], [200, 300]], np.int32)
cv2.fillPoly(img, [pts], (255, 255, 255)) # 畫白色實心多邊形

# 2. 轉為灰階並二值化
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
_, thresh = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)

# 3. 尋找輪廓
contours, _ = cv2.findContours(thresh, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

for cnt in contours:
    # 畫出輪廓本身 (綠色線條，粗度 2)
    cv2.drawContours(img, [cnt], -1, (0, 255, 0), 2)
    
    # A. 正外接矩形 (黃色)
    x, y, w, h = cv2.boundingRect(cnt)
    cv2.rectangle(img, (x, y), (x + w, y + h), (0, 255, 255), 2)
    
    # B. 最小外接矩形 (紅色)
    rect = cv2.minAreaRect(cnt)
    box = cv2.boxPoints(rect)
    box = np.int0(box)
    cv2.drawContours(img, [box], 0, (0, 0, 255), 2)
    
    # C. 最小外接圓 (藍色)
    (cx, cy), radius = cv2.minEnclosingCircle(cnt)
    cv2.circle(img, (int(cx), int(cy)), int(radius), (255, 0, 0), 2)

# 4. 顯示結果
cv2.imshow('Contour Detection Demo', img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
