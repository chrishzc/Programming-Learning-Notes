# 15. 透視變換與影像校正

> [!ABSTRACT]
> 本章介紹 OpenCV 中的透視變換（Perspective Transform）技術。透視變換常被應用於影像校正、三維投影重組等實務場景（例如將歪斜的文件或車牌拉直）。本章將解析仿射與透視變換的本質差異、說明核心 API 的呼叫方式，並提供完整的文件掃描傾斜拉直實戰程式碼。

---

## 一、 仿射變換與透視變換的區別

在二維影像的幾何轉換中，主要分為仿射變換與透視變換：

- **仿射變換 (Affine Transform)**：
  - **特徵**：轉換後，原本平行的線條**依然保持平行**。
  - **控制點**：只需要輸入 **3 個點**的對應關係即可計算出變換矩陣（自由度為 6）。
  - 常見類型包括平移、縮放、旋轉與剪切（Shear）。
- **透視變換 (Perspective Transform / Homography)**：
  - **特徵**：轉換後，原本平行的線條**不一定保持平行**，但直線在轉換後依然是直線。這是一種將二維影像投影到新視角平面（三維投影空間）的技術。
  - **控制點**：需要輸入 **4 個點**（任何三點不可共線）的對應關係來計算透視變換矩陣（自由度為 8）。
  - 常用於修正因相機拍攝角度傾斜而造成的物體形變。

---

## 二、 透視變換核心 API

OpenCV 提供了兩個核心函數來實現透視變換：

### 1. 計算透視變換矩陣
```python
M = cv2.getPerspectiveTransform(src, dst)
```
- **`src`**：輸入影像中的 4 個頂點座標，格式為 `np.float32` 的 $4 \times 2$ 陣列。
- **`dst`**：輸出（目標）影像中的 4 個對應頂點座標，格式同樣為 `np.float32` 的 $4 \times 2$ 陣列。
- **`M`**：回傳的 $3 \times 3$ 透視變換矩陣。

### 2. 套用透視變換
```python
dst_img = cv2.warpPerspective(src, M, dsize, flags=cv2.INTER_LINEAR, borderMode=cv2.BORDER_CONSTANT, borderValue=0)
```
- **`src`**：原始輸入影像。
- **`M`**：$3 \times 3$ 的透視變換矩陣。
- **`dsize`**：輸出影像的尺寸大小，格式為 `(寬度, 高度)`。
- **`flags`**：插值方法，預設為 `cv2.INTER_LINEAR`。
- **`borderMode`**：邊界填充模式。
- **`borderValue`**：邊界填充顏色，預設為黑色 (`0`)。

## 三、 實戰演練：歪斜文件拉直校正

以下程式碼示範如何讀取一張傾斜拍攝的卡片/文件影像，定義其四個角落頂點，並利用透視變換將其拉直成標準的矩形影像：

```python
import cv2
import numpy as np

# 1. 讀取原始影像
img = cv2.imread("tilted_document.jpg")
h, w = img.shape[:2]

# 2. 定義傾斜文件的四個角落座標 (順序：左上、右上、右下、左下)
# 備註：在實際專案中，這四個點通常會透過邊緣檢測與輪廓尋找演算法自動定位
src_pts = np.float32([
    [150, 200],   # 左上點
    [650, 120],   # 右上點
    [800, 700],   # 右下點
    [50, 650]     # 左下點
])

# 3. 定義轉換後標準矩形文件的寬度與高度
out_w = 500
out_h = 350

# 4. 定義目標影像中的四個對應頂點座標
dst_pts = np.float32([
    [0, 0],             # 左上
    [out_w - 1, 0],     # 右上
    [out_w - 1, out_h - 1], # 右下
    [0, out_h - 1]      # 左下
])

# 5. 計算 $3 \times 3$ 透視變換矩陣
M = cv2.getPerspectiveTransform(src_pts, dst_pts)

# 6. 進行透視變換（拉直影像）
corrected_img = cv2.warpPerspective(img, M, (out_w, out_h))

# 7. 顯示結果
cv2.imshow("Original Image", img)
cv2.imshow("Corrected Document", corrected_img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
