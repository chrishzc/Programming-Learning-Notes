# OpenCV 數學形態學運算

> [!ABSTRACT]
> 本篇筆記介紹數位影像處理中的數學形態學（Mathematical Morphology）。內容涵蓋基礎的侵蝕（Erosion）與膨脹（Dilation）、進階的複合運算（開、閉、梯度、頂帽、黑帽運算，使用 `cv2.morphologyEx`），以及如何建立自訂結構核心（`cv2.getStructuringElement`）。

---

## 1. 形態學運算基礎

形態學操作是根據影像形狀特徵進行的像素邏輯運算，**通常作用於二值化影像（Binary Images）**上。
進行形態學操作需要兩個核心輸入：
1. **原始二值影像**：白色（`255`）為前景目標物，黑色（`0`）為背景。
2. **結構元素 (Structuring Element / Kernel)**：代表滑動卷積核的形狀與大小，決定了運算對影像邊緣的形狀約束。

---

## 2. 基礎運算：侵蝕與膨脹

### 2.1 侵蝕運算 (Erosion)
- **原理**：當 Kernel 滑動覆蓋的影像區域內**所有像素皆為白色 (255)** 時，中心像素才保留為白色；否則（只要有一個像素為黑色），中心像素即變為黑色 (0)。
- **效果**：影像中靠著邊緣的像素會被「剝蝕」掉，使**前景物體縮小**、細線變細或斷開。
- **主要用途**：消除影像中微小零散的白色噪點（孤立點）、分離原本黏連在一起的物體。

#### 語法：
```python
dst = cv2.erode(src, kernel[, dst[, anchor[, iterations[, borderType[, borderValue]]]]])
```
- `iterations`：侵蝕的重複執行次數（預設為 1）。次數越多，侵蝕效果越強烈。

---

### 2.2 膨脹運算 (Dilation)
- **原理**：當 Kernel 滑動覆蓋的影像區域內**只要有一個像素為白色 (255)**，中心像素即變為白色 (255)。
- **效果**：影像中靠著邊緣的像素會向外「擴張」，使**前景物體增大**、細小孔洞或裂縫被填滿。
- **主要用途**：在去除噪點後恢復物體的大小、連結原本因斷裂而分離的物體筆畫。

#### 語法：
```python
dst = cv2.dilate(src, kernel[, dst[, anchor[, iterations[, borderType[, borderValue]]]]])
```

---

## 3. 進階形態學運算 (`cv2.morphologyEx`)

透過將侵蝕與膨脹以不同順序組合，或與原圖進行差分運算，可以達到多種精妙的去噪與特徵提取功能。

### 3.1 語法
```python
dst = cv2.morphologyEx(src, op, kernel[, iterations])
```
- **`op`**：形態學操作的類型常數（如下表所示）。

### 3.2 複合運算模式對照表
| 運算類型 (op 常數) | 動作順序 | 核心目標與物理涵義 | 最佳應用場景 |
| :--- | :--- | :--- | :--- |
| **開運算**<br>`cv2.MORPH_OPEN` | **先侵蝕，後膨脹** | 消除前景物件外部邊緣的突起與**孤立小雜訊**。 | 斷開細微連接處、去除影像「毛邊」。 |
| **閉運算**<br>`cv2.MORPH_CLOSE` | **先膨脹，後侵蝕** | 填補前景物件內部的**微小孔洞、裂縫**。 | 黏合文字中斷開的筆畫、封閉小缺口。 |
| **形態學梯度**<br>`cv2.MORPH_GRADIENT` | **膨脹圖 $-$ 侵蝕圖** | 消除所有實心面積，僅保留像素變化劇烈的邊界。 | **提取物體輪廓線**（外框）。 |
| **頂帽運算 (Top Hat)**<br>`cv2.MORPH_TOPHAT` | **原圖 $-$ 開運算圖** | 挖出影像中比周圍環境更亮、且尺寸小於 Kernel 的微小特徵。 | **黑背景中提取白色亮點**，或改善不均勻光照的背景。 |
| **黑帽運算 (Black Hat)**<br>`cv2.MORPH_BLACKHAT` | **閉運算圖 $-$ 原圖** | 挖出影像中比周圍環境更暗、且尺寸小於 Kernel 的微小特徵。 | **白背景中提取黑色斑點/細紋**。 |

---

## 4. 結構元素核之形狀設定 (`cv2.getStructuringElement`)

除了使用 NumPy 建立全 1 矩陣外，OpenCV 提供了專屬 API 來動態生成不同形狀的 Kernel：

```python
kernel = cv2.getStructuringElement(shape, ksize)
```
- **`shape`**：核心形狀常數。
- **`ksize`**：核心尺寸大小 `(寬, 高)`，必須是奇數對，如 `(5, 5)`。

### 4.1 核心形狀常數對照表
| 核心形狀 | 常數名稱 (Shape) | 視覺物理特效 | 最佳應用場景 |
| :--- | :--- | :--- | :--- |
| **矩形** | `cv2.MORPH_RECT` | 結構**方正、硬朗**。保留並強化水平與垂直的直角。 | 識別表格線條、身分證邊框、條碼。 |
| **十字形** | `cv2.MORPH_CROSS` | 帶有**十字方向性**。僅對正上、下、左、右方向有反應。 | 提取地圖路網格線、過濾斜向雜訊。 |
| **橢圓形** | `cv2.MORPH_ELLIPSE` | 邊緣**圓潤、平滑**。符合自然物體的物理輪廓。 | 去除細胞、硬幣、彈珠等圓形前景的雜訊。 |

---

## 5. 形態學運算完整實作範例

以下程式碼建立一個包含雜訊與孔洞的測試影像，並展示多種形態學處理的對比：

```python
import cv2
import numpy as np

# 1. 建立一個 300x300 的測試二值化影像 (黑色背景)
img = np.zeros((300, 300), dtype=np.uint8)
# 在中心畫一個白色實心方塊
cv2.rectangle(img, (100, 100), (200, 200), 255, -1)
# 加上前景外的白色小雜訊 (開運算準備消除)
img[50, 50] = 255
img[60, 240] = 255
# 加上方塊內部的黑色小孔洞 (閉運算準備填補)
img[150, 150] = 0

# 2. 建立 5x5 的矩形結構元素
kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (5, 5))

# 3. 執行基礎與進階形態學操作
eroded = cv2.erode(img, kernel, iterations=1)
dilated = cv2.dilate(img, kernel, iterations=1)
opened = cv2.morphologyEx(img, cv2.MORPH_OPEN, kernel)
closed = cv2.morphologyEx(img, cv2.MORPH_CLOSE, kernel)
gradient = cv2.morphologyEx(img, cv2.MORPH_GRADIENT, kernel)

# 顯示所有結果進行對比
cv2.imshow('Original with Noise', img)
cv2.imshow('Erosion', eroded)
cv2.imshow('Dilation', dilated)
cv2.imshow('Open (Remove Noise)', opened)
cv2.imshow('Close (Fill Hole)', closed)
cv2.imshow('Morph Gradient (Outline)', gradient)

cv2.waitKey(0)
cv2.destroyAllWindows()
```
