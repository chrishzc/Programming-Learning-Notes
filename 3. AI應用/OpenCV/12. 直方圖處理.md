# OpenCV 影像直方圖處理與均衡化

> [!ABSTRACT]
> 本篇筆記介紹影像直方圖（Histogram）的計算方法，如何透過 Matplotlib 繪製直方圖，以及如何使用直方圖均衡化（全域均衡化與自適應局部均衡化 CLAHE）來增強影像對比度。

---

## 1. 影像直方圖概念

影像直方圖是統計影像中各個亮度值（`0 ~ 255`）像素出現頻率的統計圖。
- **X 軸**：像素強度值（`0` 代表最暗，`255` 代表最亮）。
- **Y 軸**：該強度值像素在影像中的總個數。
- **作用**：直觀觀察影像的曝光程度、對比度高低，作為後續選擇閥值或影像增強演算法的依據。

---

## 2. 計算直方圖 (`cv2.calcHist`)

OpenCV 提供 `cv2.calcHist` 函數來統計影像的像素強度分佈。

### 2.1 語法
```python
hist = cv2.calcHist(images, channels, mask, histSize, ranges)
```

### 2.2 參數說明
| 參數名稱 | 資料型態 | 參數說明與設定規則 |
| :--- | :--- | :--- |
| **`images`** | `list` | 輸入影像 List。型態必須為 `uint8` 或 `float32`，變數**必須用中括號包起來**，例如：`[img]`。 |
| **`channels`** | `list` | 指定分析通道 List，**必須用中括號包起來**。例：灰階影像為 `[0]`；彩色影像若要分析 Blue、Green、Red 則分別為 `[0]`、`[1]`、`[2]`。 |
| **`mask`** | `ndarray` | 遮罩影像。設為 `None` 代表分析整張影像；若傳入二值遮罩，則僅統計白色區域（`255`）的像素。 |
| **`histSize`** | `list` | 區段個數（Bins）。即長條圖中要切分成幾條直條。**必須用中括號包起來**，統計 `0 ~ 255` 完整亮度則設為 `[256]`。 |
| **`ranges`** | `list` | 統計的像素值區間。**必須用中括號包起來**，通常設為 `[0, 256]`（包含 `0` 到 `255` 的值）。 |

---

## 3. 直方圖均衡化 (Histogram Equalization)

直方圖均衡化是一種增強影像對比度的方法，藉由將分布集中的像素亮度範圍「拉伸」並均勻鋪開，以改善影像的視覺品質。

| 處理技術 | 語法實作 | 運作邏輯 | 核心缺點與適用場景 |
| :--- | :--- | :--- | :--- |
| **全域均衡化**<br>(Global) | `cv2.equalizeHist()` | 算出整張影像的總體亮度分佈，以相同力道將整張影像的動態範圍拉開。 | ❌ **缺點**：若局部有強光或偏亮（如正常天空），會被強行放大而過曝。<br><br>👉 適合**光源極度均勻、但整體偏暗**的影像。 |
| **自適應局部均衡化**<br>(CLAHE) | `cv2.createCLAHE()` | **因地制宜**。將影像分割成多個 $8 \times 8$ 的局部小方塊（Tiles）各自進行均衡化。 | ⭐️ **首選王牌！** 內建「限幅（Clip Limit）」機制，若某小塊反光過強，會將多餘部分削掉以保護細節。 <br><br>👉 適合**光影複雜、有局部陰影與強光**的複雜場景。 |

---

## 4. 直方圖處理實戰範例

### 4.1 範例 A：計算並使用 Matplotlib 繪製彩色直方圖
```python
import cv2
import matplotlib.pyplot as plt

# 載入彩色影像
img = cv2.imread('input.jpg')

# 統計三通道色彩並繪製
colors = ('b', 'g', 'r')
plt.figure(figsize=(8, 5))
plt.title("BGR Color Histogram")
plt.xlabel("Bins (Pixel Intensity)")
plt.ylabel("# of Pixels")

for i, col in enumerate(colors):
    # 計算單一通道直方圖
    hist = cv2.calcHist([img], [i], None, [256], [0, 256])
    # 使用對應色彩繪線
    plt.plot(hist, color=col)
    plt.xlim([0, 256])

plt.show()
```

---

### 4.2 範例 B：全域均衡化與 CLAHE 對比
```python
import cv2

# 載入灰階影像
gray_img = cv2.imread('low_contrast.jpg', cv2.IMREAD_GRAYSCALE)

# 1. 執行全域直方圖均衡化
global_equalized = cv2.equalizeHist(gray_img)

# 2. 執行自適應局部直方圖均衡化 (CLAHE)
# 建立 CLAHE 物件，限制對比度閾值為 2.0，局部區塊為 8x8
clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8, 8))
clahe_result = clahe.apply(gray_img)

# 顯示對比結果
cv2.imshow('Original Gray', gray_img)
cv2.imshow('Global Equalized', global_equalized)
cv2.imshow('CLAHE Result', clahe_result)

cv2.waitKey(0)
cv2.destroyAllWindows()
```
