# Dlib人臉偵測、OpenCV哈爾級聯與OCR文字識別

> [!ABSTRACT]
> 本篇筆記介紹如何使用 Dlib 函式庫進行人臉檢測與 68 個面部特徵點定位、利用 OpenCV 的哈爾級聯分類器（Haar Cascade Classifier）偵測五官，以及使用 Pytesseract 套件進行圖像文字光學辨識（OCR）的實作方法。

---

## 1. Dlib 影像辨識與人臉特徵點偵測

Dlib 是一套整合機器學習與影像處理的 C++ 開源函式庫，並提供了高效率的 Python API，廣泛應用於人臉偵測與定位。

### 1.1 Dlib 正臉偵測器
Dlib 內建了以 HOG（Histogram of Oriented Gradients）特徵結合線性分類器（SVM）訓練的正臉偵測器，速度快且精準度高。
- **Unsample 參數**：`detector(img, 1)`。第二個參數代表「影像上採樣（放大）的次數」。當相片中的人臉尺寸偏小或解析度較低時，設為 `1` 或更高可以提高小人臉的偵測成功率。

### 1.2 Dlib 68 點面部關鍵點檢測 (Facial Landmarks)
Dlib 提供了一個基於預訓練模型（`shape_predictor_68_face_landmarks.dat`）的特徵點檢測器。它能精確定位出人臉五官（眉毛、眼睛、鼻子、嘴巴、下巴輪廓）共 68 個關鍵點的座標。

#### 68 點特徵點標記實作範例：
> [!IMPORTANT]
> 執行此範例前，請確保已在程式執行目錄下放置了 `shape_predictor_68_face_landmarks.dat` 權重模型檔案。

```python
import cv2
import dlib

# 1. 初始化人臉偵測器與 68 點特徵點預測器
detector = dlib.get_frontal_face_detector()
predictor = dlib.shape_predictor('shape_predictor_68_face_landmarks.dat')

# 2. 讀取彩色影像並轉為灰階 (偵測常在灰階下執行)
img = cv2.imread('face.jpg')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 3. 偵測影像中的所有人臉框
face_rects = detector(gray, 1)
print(f"偵測到的人臉數量: {len(face_rects)}")

for i, d in enumerate(face_rects):
    # A. 取得人臉邊框座標並繪製綠框 (left, top, right, bottom)
    x1, y1, x2, y2 = d.left(), d.top(), d.right(), d.bottom()
    cv2.rectangle(img, (x1, y1), (x2, y2), (0, 255, 0), 2)
    cv2.putText(img, f'Face {i}', (x1, y1 - 10), cv2.FONT_HERSHEY_SIMPLEX, 0.7, (0, 255, 0), 2)
    
    # B. 預測該人臉的 68 個特徵點
    landmarks = predictor(gray, d)
    
    # 走訪 68 個特徵點並在畫面上標記紅色實心小圓圈
    for n in range(0, 68):
        x = landmarks.part(n).x
        y = landmarks.part(n).y
        cv2.circle(img, (x, y), 2, (0, 0, 255), -1)

# 顯示特徵點標記結果
cv2.imshow('Dlib 68 Landmarks Detection', img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## 2. OpenCV 哈爾特徵級聯分類器 (Haar Cascades)

哈爾級聯分類器是 OpenCV 中經典的機器學習物件偵測演算法，常用於偵測人臉、眼睛、微笑、貓臉等。

### 2.1 內建模型清單 (XML)
OpenCV 官方提供了許多預先訓練好的分類器模型 XML 檔案，常用包括：
- `haarcascade_frontalface_default.xml`：正面人臉偵測。
- `haarcascade_eye.xml`：眼睛偵測。
- `haarcascade_eye_tree_eyeglasses.xml`：戴眼鏡的眼睛偵測。

### 2.2 偵測方法說明
```python
faces = face_cascade.detectMultiScale(gray, scaleFactor, minNeighbors, minSize)
```
- **`scaleFactor`**：影像縮放比例（大於 1，例如 `1.1`）。演算法會在不同尺寸的金字塔上掃描，值越小越精細但計算越慢。
- **`minNeighbors`**：特徵點的相鄰合併閥值。值越大，判定越嚴格（減少誤判），但可能會漏檢。
- **`minSize`**：偵測物件的最小尺寸（例如限制小於 `30x30` 的雜訊點不予偵測）。

#### 臉部與眼部雙重偵測實作範例：
```python
import cv2

# 1. 載入 OpenCV 內建的 Haar 級聯分類器模型
# 在 Python 中，可直接透過 cv2.data.haarcascades 取得內建 XML 模型路徑
face_cascade = cv2.CascadeClassifier(cv2.data.haarcascades + 'haarcascade_frontalface_default.xml')
eye_cascade = cv2.CascadeClassifier(cv2.data.haarcascades + 'haarcascade_eye.xml')

# 2. 讀取影像並轉為灰階
img = cv2.imread('face_image.jpg')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 3. 偵測人臉
faces = face_cascade.detectMultiScale(gray, scaleFactor=1.1, minNeighbors=5, minSize=(30, 30))

for (x, y, w, h) in faces:
    # 繪製人臉綠色框
    cv2.rectangle(img, (x, y), (x + w, y + h), (0, 255, 0), 2)
    
    # 限制眼睛的偵測範圍僅在「人臉框區域 (ROI)」內，以提升速度並過濾背景干擾
    roi_gray = gray[y:y+h, x:x+w]
    roi_color = img[y:y+h, x:x+w]
    
    # 在人臉區域內偵測眼睛
    eyes = eye_cascade.detectMultiScale(roi_gray, scaleFactor=1.1, minNeighbors=10)
    for (ex, ey, ew, eh) in eyes:
        # 在人臉區域內繪製藍色眼睛框
        cv2.rectangle(roi_color, (ex, ey), (ex + ew, ey + eh), (255, 0, 0), 2)

cv2.imshow('Haar Cascade Detection', img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## 3. Pytesseract 光學字元識別 (OCR)

OCR（Optical Character Recognition，光學字元識別）是指將影像中的文字線條轉換為可編輯之電腦字串文字的技術。
- **Tesseract**：由 Google 贊助與維護的開源 OCR 引擎，是目前公認最優秀、最精準的開源 OCR 系統。
- **Pytesseract**：Tesseract 引擎的 Python 包裝器（Wrapper）。

#### 影像文字辨識實作範例：
> [!WARNING]
> **Windows 環境特別提示**
> 若您在 Windows 下執行 pytesseract 遭遇 `TesseractNotFoundError`，代表系統找不到 Tesseract 主程式。
> 您必須手動在程式中指定 `tesseract.exe` 的安裝路徑：
> ```python
> import pytesseract
> pytesseract.pytesseract.tesseract_cmd = r'C:\Program Files\Tesseract-OCR\tesseract.exe'
> ```

```python
import cv2
import pytesseract

# 1. 讀取文字影像並轉為灰階
img = cv2.imread('printed_text.jpg')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 2. 進行二值化處理以消除雜訊、增強文字邊緣，能大幅提高 OCR 辨識準確率
_, thresh = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)

# 3. 使用 pytesseract 識別英文文字
text_english = pytesseract.image_to_string(thresh, lang='eng')
print("--- 英文 OCR 識別結果 ---")
print(text_english.strip())
```
