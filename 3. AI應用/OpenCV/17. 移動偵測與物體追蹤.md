# 17. 移動偵測與物體追蹤

> [!ABSTRACT]
> 本章介紹 OpenCV 中的移動偵測（Motion Detection）與視訊物體追蹤（Object Tracking）技術。內容涵蓋訊號相減法、高斯混合背景消除模型（MOG2）、經典的 Meanshift 與 CamShift 色彩追蹤演算法，以及光流法（Optical Flow）的核心觀念，並提供基於 MOG2 演算法的即時移動偵測實戰程式碼。

---

## 一、 訊號相減法與背景消除 (Frame Differencing vs Background Subtraction)

在視訊串流中，偵測移動物體最常用的方法有兩種：

- **訊號相減法 (Frame Differencing)**：
  - **原理**：將當前影格與前一影格（或背景影格）直接進行矩陣相減。像素差值大於一定閾值的區域即判定為移動區域。
  - **優缺點**：計算速度極快，但極易受到光照變更、攝影機微抖動及噪聲干擾。
- **背景消除法 (Background Subtraction / MOG2)**：
  - **原理**：利用混合高斯模型（MOG2）為場景中的背景建立統計模型。隨著時間推移，背景模型會自我更新，從而排除樹枝晃動、光影變化等干擾，精確分離出前景（移動目標）。
  - **Keras/OpenCV API**：
    ```python
    fgbg = cv2.createBackgroundSubtractorMOG2(history=500, varThreshold=16, detectShadows=True)
    fgmask = fgbg.apply(frame) # 產生前景遮罩 (白色為移動物體)
    ```

---

## 二、 均值漂移與自適應均值漂移 (Meanshift & CamShift)

當我們想要在視訊中鎖定並追蹤一個特定物體（例如追蹤一個紅色球體）時，可以使用基於特徵分佈的追蹤演算法：

- **Meanshift (均值漂移)**：
  - **原理**：首先定義追蹤目標的色彩直方圖（通常在 HSV 空間）。在下一影格中，計算色彩反向投影，演算法會在初始設定的搜尋視窗內迭代計算像素密度中心（質心），並將視窗「漂移」到質心，直到收斂。
  - **局限性**：追蹤視窗的**大小固定**。如果物體靠近或遠離攝影機，視窗無法隨物體尺寸變化而縮放。
- **CamShift (Continuously Adaptive Meanshift)**：
  - **原理**：Meanshift 的進階版。它在每次 Meanshift 收斂後，會自動計算目標區域的面積大小與方向，並**動態調整追蹤視窗的尺寸與旋轉角度**。

## 三、 光流法 (Optical Flow)

光流法是用於估計連續影格之間像素點運動方向與速度的技術。

- **核心假設**：
  1. 相鄰影格之間的物體像素亮度值（強度）保持不變。
  2. 相鄰像素點之間的運動速度與方向相似（空間一致性）。
- **分類**：
  - **稀疏光流 (Sparse Optical Flow / Lucas-Kanade)**：只計算影像中特定特徵點（如角點）的運動軌跡。計算速度快，常用於特徵追蹤。
    * OpenCV API: `cv2.calcOpticalFlowPyrLK()`
  - **稠密光流 (Dense Optical Flow / Farneback)**：計算影像中所有像素點的運動向量。常用於計算全場景動態與運動光流圖，但計算量極大。
    * OpenCV API: `cv2.calcOpticalFlowFarneback()`

## 四、 實戰演練：MOG2 即時移動偵測

以下程式碼示範如何使用 MOG2 背景消除器，在攝影機視訊畫面中即時識別移動物體，並自動用綠色矩形框（Bounding Box）將其標記出來：

```python
import cv2

# 1. 開啟預設攝影機
cap = cv2.VideoCapture(0)

# 2. 建立 MOG2 背景消除器
fgbg = cv2.createBackgroundSubtractorMOG2(history=500, varThreshold=25, detectShadows=True)

while True:
    ret, frame = cap.read()
    if not ret:
        break
        
    # 3. 套用背景消除器取得前景遮罩
    fg_mask = fgbg.apply(frame)
    
    # 4. 進行形態學處理消除微小噪聲 (開運算 ➔ 閉運算)
    kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (5, 5))
    fg_mask = cv2.morphologyEx(fg_mask, cv2.MORPH_OPEN, kernel)
    fg_mask = cv2.morphologyEx(fg_mask, cv2.MORPH_CLOSE, kernel)
    
    # 5. 尋找前景遮罩中的移動物體輪廓
    contours, _ = cv2.findContours(fg_mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    
    for contour in contours:
        # 過濾太小的噪聲輪廓 (面積小於 1000 像素)
        if cv2.contourArea(contour) < 1000:
            continue
            
        # 6. 計算輪廓的包圍外接矩形
        x, y, w, h = cv2.boundingRect(contour)
        
        # 在原始畫面中繪製綠色框
        cv2.rectangle(frame, (x, y), (x + w, y + h), (0, 255, 0), 2)
        cv2.putText(frame, "Moving Object", (x, y - 10), 
                    cv2.FONT_HERSHEY_SIMPLEX, 0.5, (0, 255, 0), 1)

    # 7. 顯示視訊畫面與前景遮罩
    cv2.imshow("Real-time Motion Detection", frame)
    cv2.imshow("MOG2 Foreground Mask", fg_mask)
    
    # 按 'q' 鍵退出
    if cv2.waitKey(30) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()
```
