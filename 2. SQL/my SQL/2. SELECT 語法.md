# SELECT 基礎查詢語法

> [!ABSTRACT]
> 學習使用 SELECT 進行基礎資料查詢，包含欄位選取、別名設定、去重 (DISTINCT) 與基礎算術運算。

---

## 1. 基本語法結構
```sql
SELECT <欄位名稱> FROM <資料表名稱>;
```
- **逗號區隔**：若選取多個欄位，中間需以逗號 `,` 分隔。
- **結束符號**：SQL 敘述結尾必須加上分號 `;`。

### 1.1 查看表格結構
在查詢之前，通常會先確認欄位名稱與型態：
```sql
DESCRIBE <table>; -- 或縮寫 DESC <table>;
```

---

## 2. 查詢目標 (Target) 分類
1. **欄位 (Column Name)**：
   - 不分大小寫，但查詢結果的表頭會維持查詢時的大小寫。
   - 使用 `*` 代表選取所有欄位。
2. **常值 (Literal)**：固定的字串、數字或日期（非表格內的資料）。
3. **運算式 (Expression)**：支援算術運算（先乘除後加減）。可以直接對數值類型的欄位進行運算。
4. **函數 (Function)**：如 `CONCAT()`, `DISTINCT`, `LIMIT` 等。

---

## 3. 欄位別名 (Column Alias)
用於美化查詢結果的表頭。
```sql
SELECT <column> [AS] Alias_Name FROM <table>;
```
- **AS 關鍵字**：可省略。
- **引號使用**：別名若包含**空白**或**特殊字元**，必須使用單引號 `' '` 括起來。

## 4. 特殊查詢技巧

### 4.1 處理空值 (Null Value)
> [!WARNING]
> 空值 (NULL) 代表「未知」或「不存在」。**NULL 無法直接參與運算**，任何數值與 NULL 運算的結果皆為 **NULL**。

### 4.2 字串連結 (Concatenation)
使用 `CONCAT()` 函數將多個值串接在一起：
```sql
CONCAT(expr1, expr2, ...); -- expr 可以是任何資料型態
```
![[{D8B142E0-F0C1-4729-8ECF-3D91A7084F45}.png|481]]

### 4.3 排除重複資料 (Distinct)
```sql
SELECT DISTINCT <column> FROM <table>;
```

## 5. 限制回傳筆數 (Limit)
用於分頁或僅取前幾筆資料。
```sql
SELECT <columns>
FROM <table>
LIMIT [offset,] count;
```
- **count**：要回傳的最大資料筆數。
- **offset**（選填）：略過前面幾筆資料。例如 `LIMIT 5, 10` 代表略過前 5 筆，取出後面的 10 筆。
