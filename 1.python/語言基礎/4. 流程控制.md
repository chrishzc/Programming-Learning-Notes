# 4. 流程控制

> [!ABSTRACT]
> 掌握程式的邏輯走向，包含條件判斷（if-else, match-case）與迴圈控制（for, while），讓程式具備決策能力。

---

## 一、 條件判斷
### 1. if-else 敘述句
Python 使用 **縮排 (Indentation)** 來區隔程式區塊。

> [!TIP]
> **Truthy 與 Falsy 觀念**：在 Python 的 `if` 條件中，不一定要放入布林值。
> - 以下情況會被當作 **False (Falsy)**：`0`, `0.0`, `""` (空字串), `[]` (空串列), `{}` (空字典), `None`, `False`。
> - 其他所有非空值都會被當作 **True (Truthy)**。
> 利用這個特性可以寫出非常簡潔的判斷式（例如：`if not my_list:` 來判斷串列是否為空）。

```python
# 1. if-elif-else 與 Truthy/Falsy 範例
score = 75

if score >= 90:
    print("優秀")
elif score >= 60:
    print("及格")
else:
    print("不及格")
# 輸出: 及格

# Truthy / Falsy 應用
my_name = ""
if not my_name:
    print("姓名尚未輸入！") # 空字串是 Falsy，加上 not 變成 True，因此會執行
```

### 2. 條件運算式 (Ternary Operator)
簡潔的一行判斷寫法：
`變數 = x if 條件 else y`

```python
# 2. 條件運算式範例
age = 16
status = "可以投票" if age >= 18 else "不能投票"
print(status) # 輸出: 不能投票
```

### 3. match-case (Python 3.10+)
適用於多重等值比對。
```python
# 3. match-case 範例
status_code = 404

match status_code:
    case 200:
        print("連線成功")
    case 400 | 404: # 使用 | 代表 OR
        print("客戶端錯誤")
    case 500:
        print("伺服器錯誤")
    case _:        # 相當於 default
        print("未知狀態")
# 輸出: 客戶端錯誤
```

## 二、 迴圈控制
### 1. while 迴圈
當條件為 `True` 時持續執行。
> [!TIP]
> **無窮迴圈的設計技巧**：常用 `while True:` 配合 `break` 建立無窮迴圈，直到特定結束條件達成時才跳出。
> 
> ```python
> # 1. while 迴圈與無限迴圈範例
> count = 1
> while count <= 3:
>     print(f"Count: {count}")
>     count += 1
> 
> # 無限迴圈配 break 範例
> while True:
>     answer = "yes" # 模擬使用者輸入
>     if answer == "yes":
>         print("結束無限迴圈")
>         break
> ```

### 2. for 迴圈
遍歷固定次數或序列資料。
- **range(start, stop, step)**: 產生的數列包含 start，但 **不包含 stop**。

```python
# 2. for 迴圈與 range() 範例
# 產生 1, 2, 3, 4 (不包含 5)
for i in range(1, 5):
    print(i, end=" ") # 輸出: 1 2 3 4
print() # 換行

# 指定間隔值 (step)
# 產生 0, 2, 4
for i in range(0, 6, 2):
    print(i, end=" ") # 輸出: 0 2 4
print()
```

## 三、 特殊流程處理
- **break**: **立刻終止** 並跳出整個迴圈。
- **continue**: **跳過本次** 剩餘程式碼，直接進入下一次迴圈。
- **迴圈的 else 語句**: `for` 或 `while` 迴圈可以搭配 `else`。當迴圈**正常結束**（完全走完且沒有被 `break` 打斷）時，就會執行 `else` 區塊。這在「尋找特定項目」的場景中非常實用。

```python
# 三、 特殊流程處理範例
for val in "string":
    if val == "i":
        continue # 當遇到 "i"，跳過此次列印，直接進入下一個字元
    if val == "g":
        break    # 當遇到 "g"，立刻中止整段 for 迴圈
    print(val, end="")
print() # 輸出: strn (跳過 i，並在 g 之前結束)

# for...else 尋找範例
search_target = 5
numbers = [1, 2, 3]

for n in numbers:
    if n == search_target:
        print("找到了！")
        break
else:
    # 只有當迴圈完整跑完，都沒有觸發 break 時，才會執行這裡
    print("清單中沒有找到目標數字。")
```
