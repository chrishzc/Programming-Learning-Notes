## 二、 常用亂數函數 (Random)

進行隨機抽樣或生成亂數時，需使用 `random` 模組：

### 1. 數值型亂數
- **`random.randint(a, b)`**：產生 $[a, b]$ 閉區間內的**整數**亂數（包含 $a$ 與 $b$）。
- **`random.randrange(start, stop, step)`**：在指定範圍內以步長產生隨機整數（不包含 $stop$）。
- **`random.uniform(a, b)`**：產生 $[a, b]$ 範圍內的隨機**浮點數**。

### 2. 序列與隨機抽樣
- **`random.choice(seq)`**：從序列中隨機選取**一個**元素。
- **`random.sample(seq, k)`**：從序列中隨機抽取 $k$ 個元素，為**不重複抽樣 (不放回)**。
- **`random.choices(seq, k)`**：從序列中隨機抽取 $k$ 個元素，**允許重複抽樣 (放回)**。
- **`random.shuffle(seq)`**：**洗牌**，隨機打亂序列的順序（直接修改原 List）。

```python
import random

# 生成隨機整數 [1, 10]
print(random.randint(1, 10))

# 從清單中隨機抽取 2 個不重複元素
fruits = ["Apple", "Banana", "Cherry", "Date"]
print(random.sample(fruits, k=2))

# 打亂清單順序
cards = [1, 2, 3, 4, 5]
random.shuffle(cards)
print(cards)  # 例如: [3, 5, 1, 4, 2]
```

- **`s.isalpha()`**：字串是否全部為「英文字母」（含有數字、空格或標點則為 `False`）。
- **`s.isdigit()`**：字串是否全部為「0~9 純數字」（含有小數點或負號則為 `False`）。

- **`s.isspace()`**：字串是否全部為「空白、Tab 或換行符」。

## 四、 常用日期與時間函數 (Datetime)

處理日期與時間需使用內建的 `datetime` 模組：

### 1. 取得當前時間與格式轉換
```python
from datetime import datetime, date

# 取得今天日期與現在時間
now = datetime.now()
print(f"現在時間: {now}")      # 例如: 2026-06-06 09:30:00.123456
print(f"年份: {now.year}, 月份: {now.month}, 星期幾: {now.weekday()}") # 星期一為 0

# 物件轉為格式化字串 (strftime: String Format Time)
time_str = now.strftime("%Y/%m/%d %H:%M:%S")
print(f"格式化文字: {time_str}") # 輸出: 2026/06/06 09:30:00

# 格式化字串轉回時間物件 (strptime: String Parse Time)
dt_obj = datetime.strptime("2000/01/01", "%Y/%m/%d")
print(dt_obj) # 輸出: 2000-01-01 00:00:00
```

### 2. 日期時間運算 (timedelta)
使用 `timedelta` 來進行時間的加減運算：
```python
from datetime import datetime, timedelta

now = datetime.now()

# 取得 7 天後的日期
future_date = now + timedelta(days=7)
print(f"7天後: {future_date.strftime('%Y-%m-%d')}")

# 計算兩個時間的差距
dt1 = datetime(2026, 6, 1)
dt2 = datetime(2026, 6, 6)
diff = dt2 - dt1
print(f"相差天數: {diff.days} 天") # 輸出: 5 天
```

## 五、 正規表示式 (Regular Expression / re)

正規表示式用於檢查文字格式是否符合特定規則，或是從大段文本中提取特定資訊。

- **`re.compile(pattern)`**：編譯正規表示式規則。
- **`pattern.fullmatch(string)`**：字串必須「完全符合」規則才算成功（常用於手機、Email 格式驗證）。
- **`pattern.search(string)`**：搜尋字串中「第一個符合」規則的子字串。
- **`pattern.findall(string)`**：找出字串中「所有符合」規則的子串並以 List 回傳。

### 常用正規表示式符號對照表
| 符號 | 說明 | 範例 |
| :--- | :--- | :--- |
| `\d` | 匹配一個 0~9 的數字 | `\d{3}` 匹配 3 位數字。 |
| `\w` | 匹配一個字母、數字或下底線 | `\w+` 匹配連續的英數單字。 |
| `[a-z]` | 匹配指定範圍內的任一字元 | `[A-Z]` 匹配大寫英文字母。 |
| `^` / `$` | 匹配字串的開頭 / 結尾 | `^09\d{8}$` 驗證台灣手機號碼格式。 |
| `?` | 前方字元出現 0 次或 1 次 | `https?` 匹配 http 或 https。 |
| `+` | 前方字元出現 1 次或多次 | `\d+` 匹配任意長度的整數。 |
| `*` | 前方字元出現 0 次或多次 | `ab*` 匹配 a, ab, abb, abbb 等。 |

### 實戰演練：手機號碼與 Email 提取
```python
import re

# 1. 驗證台灣手機號碼格式是否完全正確
phone_pattern = re.compile(r"^09\d{8}$")
print(phone_pattern.fullmatch("0912345678") is not None) # True
print(phone_pattern.fullmatch("0912345") is not None)    # False (字數不足)

# 2. 從文字中提取所有電子郵件
text = "聯絡我們: admin@example.com 或 service@google.com.tw"
email_pattern = re.compile(r"[\w\.-]+@[\w\.-]+\.\w+")
emails = email_pattern.findall(text)
print(emails) # 輸出: ['admin@example.com', 'service@google.com.tw']
```
