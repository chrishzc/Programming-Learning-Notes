# 1. 介紹與環境設置

> [!ABSTRACT]
> 認識 Python 語言的特性、與編譯式語言的差異，並完成開發環境（IDE、套件管理工具、環境變數）的基礎設置。

---

## 一、 Python 語言介紹
Python 是一種 **直譯式**、**高階**、**通用型** 程式語言。

### 1. 直譯式 vs. 編譯式
- **直譯式 (Interpreted Language)**:
    - 一邊翻譯一邊執行。
    - 寫完一小段即可立即除錯，靈活度高。
    - 代表：Python, JavaScript。
- **編譯式 (Compiled Language)**:
    - 先編譯成機器碼後再執行。
    - 執行速度通常較快。
    - 代表：C, C++, Swift。

### 2. 其他特性
- 語法簡潔，強調 **可讀性**。
- 具有完整的 **物件導向** 功能。

---

## 二、 函式庫與套件管理
**函式庫 (Libraries)** 是預先編寫好的程式碼，開發者可以直接引用，無需從零開始。

### 1. 套件平台與工具
- **PyPI (Python Package Index)**: 官方套件平台，儲存大量第三方套件。
- **pip**: 官方套件管理工具，從 PyPI 下載套件。
- **UV**: 新一代快速套件管理工具。
    - 安裝速度遠超 pip。
    - 可完全重現專案環境（包含特定版本）。

---

## 三、 環境設置
### 1. IDE (整合開發環境)
- **VS Code**: 熱門的開源編輯器，提供完善的 Coding 環境。

### 2. 系統環境變數 (PATH)
> [!IMPORTANT]
> **為什麼要設定路徑？**
> 電腦需要「預設搜尋路徑」才能找到執行檔。若指令無法執行，需將執行檔目錄加入系統的 `Path`。

> [!TIP]
> **操作方式**：在 Windows 開始搜尋「編輯系統環境變數」，並將執行檔所在目錄加入 `Path`。

### 3. 設定直譯器 (Interpreter)
在 VS Code 中需要手動指定要使用的 Python 版本。

## 四、 實戰演練：撰寫與執行第一個 Python 程式
建立一個名為 `hello.py` 的檔案，寫入以下程式碼：

```python
# 這是您的第一個 Python 程式
# 宣告字串變數並指派值
message = "哈囉，Python 語言與環境設置完成！"

# 印出結果到終端機
print(message)

# 顯示當前直譯器版本資訊 (載入內建 sys 模組)
import sys
print("當前 Python 直譯器版本：")
print(sys.version)
```

**執行步驟**：
1. 開啟終端機 (Terminal)。
2. 切換目錄至 `hello.py` 所在的資料夾下。
3. 輸入指令執行：`python hello.py`
