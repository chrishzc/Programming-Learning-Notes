# 物件導向

> [!ABSTRACT]
> 學習物件導向程式設計（OOP）的核心概念，包含類別與物件定義、建構子、魔術方法、繼承與覆寫，以及物件的搜尋、排序與 Set 集合去重應用。

---

物件導向的觀念，舉例將⼀本書以⼀個物件來表⽰，同樣要將該物件所儲存的資訊填到表格內
## 建立過程
### 1. 建立表格 :定義標題與欄位 
建立⼀筆資料以記錄⼀本書籍資訊
#### 定義類別 (class)
- **定義建構函式 `__init__`**: (官方稱為 constructor，也可稱為 initializer) 方便在建立物件時進行初始化。
- **什麼是 `self`？**: `self` 代表「當下正在呼叫或建立的這個實體物件本身」。它是類別內所有方法的第一個必備參數，讓 Python 知道現在是要對哪一個具體的物件進行操作。
- **屬性 (Attribute)**:
  - **實例屬性 (Instance Attribute)**：定義在 `__init__` 內，並加上 `self.` 前綴。每個物件都有自己獨立的一份，互不干擾。
  - **類別屬性 (Class Attribute)**：定義在所有方法之外（通常在類別的最上方），所有該類別產生出來的物件「共享」同一份資料。

```python
class Book:
    # 類別屬性 (Class Attribute)：所有 Book 物件共享
    store_name = "Python 書局"

    # 建構函式
    def __init__(self, name, price, author):
        # 實例屬性 (Instance Attribute)：每個物件獨立擁有
        # 透過 self 將資料綁定到「當前的這個物件」身上
        self.name = name
        self.price = price
        self.author = author
```
### 2.建立物件: 定義類別與屬性
建立⼀個物件以儲存⼀本書籍資訊
- 呼叫建構函式會建立物件及其實例 (instance) 
- 例如: `book = Book("Python", 500, "Paul")`

## 方法(method): 屬於定義中的函式稱作方法
類別定義的函式稱作⽅法
- ==提供給物件呼叫的⽅法第壹個參數代表現行物件，名稱習慣為 self ==
- 類別內不能重複定義⽅法 (名稱不可相同)

```python
class Book:
    def __init__(self, name, price, author):
        self.name = name
        self.price = price
        self.author = author

    # 定義方法
    def get_description(self):
        return f"書名: 《{self.name}》, 售價: {self.price}元, 作者: {self.author}"

# 建立實例並呼叫方法
book = Book("Python入門指南", 500, "Paul")
print(book.get_description()) # 輸出: 書名: 《Python入門指南》, 售價: 500元, 作者: Paul
```
## 魔術方法(Dunder methods)
為了讓自己寫的類別（Class），可以完美融入 Python 的內建語法，例如:
```python
=== 1. 物件創建與生命週期 ===
def __init__(self, ...):         # 初始化物件屬性 (最常用)
def __new__(cls, ...):           # 真正創建物件實例 (在 __init__ 之前)
def __del__(self):               # 物件被銷毀/垃圾回收時觸發
=== 2. 物件的字串與可讀性 ===
def __str__(self):               # print(obj) 或 str(obj) 時顯示的易讀字串
def __repr__(self):              # 開發者看的精準字串 (偵錯用)
=== 3. 比較運算符 ===
def __eq__(self, other):         # 關鍵字 == (Equal)
def __ne__(self, other):         # 關鍵字 != (Not Equal)
def __lt__(self, other):         # 關鍵字 <  (Less Than)
def __le__(self, other):         # 關鍵字 <= (Less or Equal)
def __gt__(self, other):         # 關鍵字 >  (Greater Than)
def __ge__(self, other):         # 關鍵字 >= (Greater or Equal)
=== 4. 數學算術運算 ===
def __add__(self, other):        # 加法 +
def __sub__(self, other):        # 減法 -
def __mul__(self, other):        # 乘法 *
def __truediv__(self, other):    # 除法 /
def __floordiv__(self, other):   # 整除 //
def __mod__(self, other):        # 取餘數 %
def __pow__(self, other):        # 次方 **
=== 5. 模擬容器/序列 (List & Dict) ===
def __len__(self):               # 呼叫 len(obj) 時觸發
def __getitem__(self, key):      # 使用中括號取值 obj[key]
def __setitem__(self, key, val): # 使用中括號賦值 obj[key] = val
def __delitem__(self, key):      # 刪除元素 del obj[key]
def __contains__(self, item):    # 使用 in 關鍵字檢查
=== 6. 上下文管理器 (With 語法) ===
def __enter__(self):             # 進入 with 區塊時觸發
def __exit__(self, t, v, tb):    # 離開 with 區塊時觸發 (釋放資源)
=== 7. 其他常用特殊操作 ===
def __call__(self, ...):         # 讓物件可以像函數一樣被呼叫 obj()
def __iter__(self):              # 讓物件可被 for 迴圈迭代
def __next__(self):              # 搭配 __iter__ 定義迭代的下一步
```

```python
# 魔術方法實作範例
class Product:
    def __init__(self, name, price):
        self.name = name
        self.price = price

    # 1. 易讀字串表達 (print(obj) 時會調用)
    def __str__(self):
        return f"{self.name} (${self.price})"

    # 2. 開發者除錯字串
    def __repr__(self):
        return f"Product(name='{self.name}', price={self.price})"

    # 3. 比較運算 (==)
    def __eq__(self, other):
        return self.price == other.price

    # 4. 加法運算 (+)
    def __add__(self, other):
        return self.price + other.price

p1 = Product("蘋果", 50)
p2 = Product("香蕉", 50)
p3 = Product("櫻桃", 100)

print(p1)                   # 輸出: 蘋果 ($50) (調用 __str__)
print(repr(p3))             # 輸出: Product(name='櫻桃', price=100) (調用 __repr__)
print(p1 == p2)             # 輸出: True (因為價格相同，調用 __eq__)
print(p1 + p3)              # 輸出: 150 (相加兩者價格，調用 __add__)
```
## 建構函式
建構函式雖然很像⽅法，但是有 3 個不同點
1. 建構函式有固定名稱，⼀般⽅法則無
	   - 建構函式名稱必須為\__init__() ，⼀般⽅法可隨意取名，只要符合識別字命名規則即可
2. 建構函式無回傳值，⼀般⽅法可有可無:
    - 建構函式主要⽬的在設定屬性初始值，所以不需要回傳值
3.  呼叫時機不同
	- 建立物件時才會呼叫建構函式；每次呼叫建構函式都會產⽣不同物件
	- 物件產⽣後才會呼叫⽅法，⽽且同⼀個物件可以呼叫多次
## 物件與 Dictionary 互轉
為了更方便使用外部套件，物件常與 dictionary 互轉
1. 物件轉成 dictionary ⽅式 
	   - \__dict__ ⽤於⼀般類別所產⽣的物件 
	   - vars() :built-in 函式，會⾃動使⽤ __dict__ ✦ 型別安全性處理較佳，可以處理較複雜情況，建議使⽤
2. dictionary 轉成物件⽅式
	   - Book(\*\*dictionary)拆解 dictionary 後利⽤建構函式建立物件

```python
# 物件與 Dictionary 互轉範例
class User:
    def __init__(self, username, role):
        self.username = username
        self.role = role

# 1. 物件轉 Dictionary (使用 vars() 或 __dict__)
user_obj = User("Alice", "Admin")
user_dict = vars(user_obj)
print(user_dict) # 輸出: {'username': 'Alice', 'role': 'Admin'}

# 2. Dictionary 轉物件 (使用 ** 拆解傳給建構子)
data = {"username": "Bob", "role": "User"}
new_user = User(**data) # 相當於 User(username="Bob", role="User")
print(new_user.username, new_user.role) # 輸出: Bob User
```
## 繼承
⼦類別 (derived class) 可以重複利⽤⽗類別 (base class) 定義的內容
### 定義⽗類別 (base class) 與實例變數
```
class Book: 
	def __init__(self, name="", price=0.0, author=""):
```
### 定義⼦類別 (derived class) 與實例變數
但在⼦類別建構函式內必須先呼叫⽗類別建構函式 
```	
class ComputerBook(Book): 
	def __init__(self, name="", price=0.0, author="", exampleUrl=""): 
		super().__init__(name, price, author) # 呼叫⽗類別 __init__() ⽅法 
		self.exampleUrl = exampleUrl   
		#self ⽤來存取所在類別的成員 
		#super 則可存取⽗類別的成員
```
### 繼承、覆寫(override)父類別方法
繼承時會自動繼承方法，但如果要修改父類別的方法，要注意==⽅法名稱要⼀致、參數個數要⼀致==
```python
#直接用一樣的方法名稱重新定義
class 子類別(父類別):
	def fathermethod()
	
#在原基礎上新增
class 子類別(父類別):
	def fathermethod()
		super().fathermethod()
		要新增的function
```

```python
# 繼承與覆寫 (Override) 完整範例
class Animal:
    def __init__(self, name):
        self.name = name

    def make_sound(self):
        return "某種動物的叫聲"

class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name) # 呼叫父類別 __init__
        self.breed = breed

    # 覆寫 (Override) 父類別的方法
    def make_sound(self):
        # 呼叫父類別的方法並延伸
        parent_sound = super().make_sound()
        return f"{parent_sound}! 汪汪！我是 {self.breed}。"

my_dog = Dog("阿奇", "柴犬")
print(my_dog.make_sound()) # 輸出: 某種動物的叫聲! 汪汪！我是 柴犬。
```
## 根類別:object 類別
是所有 Python 類別的⽗類別，是最終⽗類別 (ultimate base class)，也就是根類別 (root class)
### object 類別定義的⽅法 
- 可被其他類別覆寫
- 可被其他物件呼叫
### 隱含父類別
==如果class一開始創建時沒有特別繼承，會被python自動繼承object 類別==，同時繼承object內建函數例如\__repr__，print物件時會回傳該物件的類別以及該物件的儲存的記憶體位置。
## 物件的搜尋與排序
### 搜尋
list.index() ⽤於⾃訂物件時，需要覆寫\__eq__() 以指定欲比對的屬性(target)
```python
def __eq__(self, other): 
	return self.target == other.target
```
\__eq__()預設情況下： 如果你不覆寫這個方法，Python 在比較 b1 == b2 時，會去檢查它們是不是記憶體中的同一個物件。即使兩本書的所有資料都一模一樣，Python 也會判定為 False。

### 排序
使⽤ lambda 指定要比對的屬性(target)，預設為升冪，reverse=True 則為降冪 
```python
sorted_list = sorted(my_books, key=lambda book: book.target, reverse=True)
```

```python
# 物件搜尋與排序實作範例
class Movie:
    def __init__(self, title, rating):
        self.title = title
        self.rating = rating

    # 覆寫 __eq__ 讓 list.index() 或 in 比對時依據 title 判斷
    def __eq__(self, other):
        if not isinstance(other, Movie):
            return False
        return self.title == other.title

movies = [Movie("全面啟動", 8.8), Movie("星際效應", 8.6), Movie("黑暗騎士", 9.0)]

# 1. 搜尋物件
target = Movie("星際效應", 0.0) # rating 不影響，因為 __eq__ 只比對 title
index_found = movies.index(target)
print(f"找到該電影，索引位置為: {index_found}") # 輸出: 找到該電影，索引位置為: 1

# 2. 排序物件 (依照評分 rating 降冪排序)
sorted_movies = sorted(movies, key=lambda movie: movie.rating, reverse=True)
for m in sorted_movies:
    print(f"電影: {m.title}, 評分: {m.rating}")
# 輸出:
# 電影: 黑暗騎士, 評分: 9.0
# 電影: 全面啟動, 評分: 8.8
# 電影: 星際效應, 評分: 8.6
```
## set 類別
由於set的特性是不能放入相同值，所以==如果物件欲存放⾄ set，所屬類別必須覆寫 object 類別的 \__eq__() 與 \__hash__() ⽅法(定義值相同的定義)==，同時set 提供的數學集合函式功能可應⽤在物件上
- \_\_eq__()：明定何謂值相同
- \_\_hash__()：可以實現搜尋效率極⾼的雜湊搜尋

```python
# 物件存入 Set 的實作範例
class Student:
    def __init__(self, student_id, name):
        self.student_id = student_id
        self.name = name

    # 覆寫 __eq__ 定義屬性值相同即為相同物件
    def __eq__(self, other):
        if not isinstance(other, Student):
            return False
        return self.student_id == other.student_id

    # 覆寫 __hash__ 計算物件雜湊值，與 __eq__ 的比較欄位必須一致
    def __hash__(self):
        return hash(self.student_id)

# 建立學生集合
student_set = set()
s1 = Student(101, "Alice")
s2 = Student(101, "Alice (Duplicate)") # ID 相同，視為重複
s3 = Student(102, "Bob")

student_set.add(s1)
student_set.add(s2) # 會被自動去重
student_set.add(s3)

print(f"集合中學生數量: {len(student_set)}") # 輸出: 2
for s in student_set:
    print(f"ID: {s.student_id}, 姓名: {s.name}")
```
