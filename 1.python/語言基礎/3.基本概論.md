## 一、 變數與代碼組成
### 1. 變數宣告與指派
在許多程式語言（如 C++ 或 Java）中，開發者必須手動宣告變數類型才能使用。而 Python 採用 **動態型別 (Dynamic Typing)**：
- **動態判別**: 不需要事先宣告類型，電腦會根據你賦予的值自動推斷。
- **類型推定**: 賦值即宣告，讓代碼更簡潔。

```python
# 1. 變數宣告與指派範例
a = 1       # 自動推斷為 int (整數)
b = "Hello" # 自動推斷為 str (字串)

# 動態型別特性：同一個變數可以指向不同型態的資料
x = 100
print(f"x 的型態: {type(x)}, 值: {x}") # 輸出: x 的型態: <class 'int'>, 值: 100

x = "現在變成字串了"
print(f"x 的型態: {type(x)}, 值: {x}") # 輸出: x 的型態: <class 'str'>, 值: 現在變成字串了
```

### 2. 識別字命名規則
- **大小寫敏感**: `Apple` 與 `apple` 是不同變數。
- **首字限制**: 不能以數字開頭。
- **關鍵字限制**: 不能使用系統保留字（如 `def`, `if`）。
- **命名風格**: 建議使用 `snake_case` (如 `my_number`)。

## 二、 資料型態 (Data Types)
### 1. 字串 (str)
- **表示法**: 使用 `' '`, `" "` 或 `''' '''` (多行)。
- **跳脫字元**: `\n` (換行), `\t` (縮排), `\\` (反斜線)。
- **不可變性 (Immutable)**: 字串一旦建立後就不能修改單一字元 (如 `s[0] = 'a'` 會報錯)。任何修改都會產生一個「新的」字串。
- **常用字串方法**:
  - `upper()` / `lower()`: 轉大小寫。
  - `strip()`: 去除前後空白。
  - `replace(old, new)`: 替換字串內容。
- **字串格式化 (f-strings)**: 
    ```python
    # 1. 字串常用操作與格式化
    text = "  Hello Python  "
    print(text.strip().upper()) # 輸出: HELLO PYTHON
    print(text.replace("Python", "World").strip()) # 輸出: Hello World

    name = "Alice"
    print(f"Hello {name}") # 輸出: Hello Alice
    
    # f-string 進階格式化示範
    pi_val = 3.14159265
    print(f"圓周率保留兩位小數: {pi_val:.2f}") # 輸出: 3.14
    
    # 寬度置中對齊填充
    print(f"歡迎畫面: {'歡迎光臨':*^20}") # 輸出: *******歡迎光臨*******
    ```

### 2. 數字 (int, float)
- **算術運算子**:
  - `+` (加), `-` (減), `*` (乘), `/` (除, 結果必為浮點數)。
  - `//` (取商/向下取整), `%` (取餘數), `**` (次方)。
- **浮點數精度問題**: 比對 `float` 時盡量不要用 `==`，因為電腦儲存小數時可能存在極微小的誤差。
  
```python
# 2. 浮點數精度問題範例
a = 0.1 + 0.2
b = 0.3
print(a == b)  # 輸出: False (因為 a 的實際儲存值為 0.30000000000000004)

# 正確的比對方式：使用 math.isclose()
import math
print(math.isclose(a, b)) # 輸出: True (在允許的極小誤差內視為相等)
```
### 3. 布林值 (bool)
- 只有 `True` 與 `False` (首字母須大寫)。
- **比較運算子**: `==` (等於), `!=` (不等於), `>` (大於), `<` (小於), `>=` (大於等於), `<=` (小於等於)。比較運算的結果會回傳布林值。
- **邏輯運算子**: `and` (且), `or` (或), `not` (非/反轉)。
```python
# 3. 布林邏輯運算範例
is_weekend = True
has_free_time = False

# and: 兩者皆為 True 才為 True
print(is_weekend and has_free_time) # 輸出: False

# or: 其中一者為 True 即為 True
print(is_weekend or has_free_time)  # 輸出: True

# not: 反轉布林值
print(not is_weekend)               # 輸出: False
```

## 三、 類型轉換
- `type()`: 檢查變數型態。
- `int()`, `float()`, `str()`: 強制轉型。

```python
# 4. 類型轉換範例
x_str = "123"
x_int = int(x_str)       # 字串轉整數 -> 123
x_float = float(x_str)   # 字串轉浮點數 -> 123.0

print(type(x_int))       # 輸出: <class 'int'>
print(type(x_float))     # 輸出: <class 'float'>

# 轉為字串以進行文字串接
age = 25
print("我的年齡是: " + str(age)) # 輸出: 我的年齡是: 25
```

## 四、 輸入與輸出
### 1. print() 輸出
- `sep`: 分隔符號 (預設空白)。
- `end`: 結尾符號 (預設換行 `\n`)。
```python
# print 進階輸出範例
print("A", "B", "C", sep="-")        # 輸出: A-B-C (以 '-' 連接)
print("第一行", end="")               # 結尾不換行
print("接在第一行後面")                # 輸出: 第一行接在第一行後面
```
### 2. input() 輸入
  `input()` 接收到的資料一律為 **字串 (str)**，若要進行運算必須轉型。
  ```python
  > # input 與轉型範例
  > age_str = input("請輸入年齡: ") # 假設輸入: 25
  > # print(age_str + 1)        # 錯誤！字串不能與整數直接相加
  > age = int(age_str)          # 強制轉型為整數
  > print(f"明年你將是 {age + 1} 歲") # 輸出: 明年你將是 26 歲
  > ```
