# 檔案管理

> [!ABSTRACT]
> 學習如何使用 pathlib 與 shutil 模組進行本機目錄資訊查詢、檔案建立、複製、移動與刪除，並掌握二進位與純文字檔案的存取方法。

---

Python 提供了多種檔案管理的工具，建議使用新的 `pathlib` 套件（Python 3.5 開始支援），並搭配 `shutil` 來進行較複雜的複製與移除。

### 1. 取得目錄資訊
取得目前執行的目錄或是使用者的家目錄：
```python
from pathlib import Path

# 顯示目前執行檔案的所在目錄
current_dir = Path(__file__).resolve().parent

# 取得使用者家目錄及其父目錄
home = Path.home()
home_parent = home.parent
```
### 2. 查詢目錄或檔案的相關資訊 
透過 `Path` 物件的方法，可以輕鬆判斷路徑是檔案還是目錄：
```python
from pathlib import Path
home = Path.home()

home.is_file() # 回傳是否為檔案
home.is_dir()  # 回傳是否為目錄
home.exists()  # 回傳是否存在
```
### 3. 建立目錄或檔案 
可以使用 `/` 運算子來串接路徑，並使用 `mkdir` 和 `touch` 建立目錄與檔案：
```python
from pathlib import Path

current_dir = Path(__file__).resolve().parent
dir1 = current_dir / "test" / "dir1"
file1 = dir1 / "file1.txt"

# 建立目錄
# parents=True：路徑中所有不存在的父層目錄都會一併建立
# exist_ok=True：如果該目錄已經存在，也不會產生 FileExistsError 錯誤
dir1.mkdir(parents=True, exist_ok=True)

# 建立檔案
# 檔案不存在時會建立新的空檔案
file1.touch(exist_ok=True)
```
### 4. 檔案複製 
檔案複製需要依賴 `shutil` 套件：
```python
from pathlib import Path
import shutil

file1 = Path("test/dir1/file1.txt")
file2 = Path("test/dir1/file2.txt")

shutil.copy(file1, file2) # 將 file1 複製為 file2
```
### 5. 檔案搬移、更名 
利用 `Path` 的 `rename` 方法可以同時達到更名與搬移路徑的效果：
```python
from pathlib import Path

file1 = Path("test/dir1/file1.txt")
file3 = Path("test/file3.txt")

# 將 file1 搬移到 test 底下並改名為 file3.txt
file1.rename(file3)
```
### 6. 刪除檔案、目錄 
刪除檔案使用 `unlink()`，而刪除包含內容的整個目錄則必須使用 `shutil.rmtree()`：
```python
from pathlib import Path
import shutil

# 刪除檔案
file3 = Path("test/file3.txt")
# missing_ok=True：代表即使檔案不存在，執行時也不會發生錯誤
file3.unlink(missing_ok=True)

# 刪除整個目錄 (包含裡面所有內容)
test_dir = Path("test")
if test_dir.exists():
    shutil.rmtree(test_dir)
```
## 檔案存取導論
⼀般程式運作時，常常對 RAM 資料存取，但僅是暫存性質；如果要長久保存，最簡單⽅式就是將資料存檔。檔案內容可分成
- text：純⽂字檔案，需要考慮⽂字編碼 
- binary：⼆進位檔案 (例如：圖檔)，⼈類無法直接閱讀，需透過程式解析
## 開啟檔案存取文字檔案
文字檔案的存取可以使用 `Path` 提供的簡易方法，或是使用標準的 `with open` 進行更細部的控制。
### 1. 讀取文字檔
```python
from pathlib import Path

filePath = Path("books.txt")

# 方法一：標準讀取 (使用 with open)
# "r"(read) 代表讀資料，"t"(text) 代表文字內容；預設為 "rt"
# with 區塊結束會自動關閉 file，不需要呼叫 f.close()
# encoding (文字編碼) 預設會採用該平台的 encoding，建議明確指定
with filePath.open(mode="rt", encoding="utf-8") as f:
    for line in f:
        pass # 逐行讀出 line
    
    # 檔案相關屬性：f.name, f.closed, f.mode, f.encoding

# 方法二：簡易版讀取 (一次讀取所有文字)
# 適合檔案不大時直接讀成字串
content = filePath.read_text(encoding="utf-8")
```

### 2. 寫入文字檔
寫入文字檔時，若檔案已存在則會覆寫原有內容；若檔案不存在則會自動建立。
```python
from pathlib import Path

filePath = Path("fileOut.txt")
text = "春暖花開"

# 方法一：標準寫入 (使用 with open)
# "w" (write) 代表寫入資料，省略 "t" 預設即為文字寫入
with filePath.open(mode="w", encoding="utf-8") as f:
    f.write(text)

# 方法二：簡易版寫入
filePath.write_text(text, encoding="utf-8")
```

### 3. 附加文字至檔案後方
若希望保留檔案原本的內容，僅在最後方加入新文字，應使用 `"a"` (append) 模式。
```python
from pathlib import Path

filePath = Path("fileAppend.txt")
text = "秋高氣爽"

# "a" (append) 代表在已有的檔案內容附加文字
with filePath.open(mode="a", encoding="utf-8") as f:
    f.write(text)
```
## 存取二進位檔案 (包含圖檔)
二進位檔案（例如圖片、影音、壓縮檔等）的存取，可以使用 `"rb"` (read binary) 與 `"wb"` (write binary) 模式，或是透過簡易版的 `read_bytes()` 和 `write_bytes()` 方法。
### 1. 讀寫二進位檔 (標準分批處理)
當處理容量較大的檔案時，建議設定分批讀取的資料量（chunk size），避免一次讀取佔用過多記憶體。
```python
from pathlib import Path

source = Path("image.png")
destination = Path("binary.png")

# 怕檔案過大，設定每次讀入並暫存的資料量 (例如: 1024 bytes)
chunk_size = 1024

# "rb" 讀取二進位，"wb" 寫入二進位
with source.open(mode="rb") as fs, destination.open(mode="wb") as fd:
    while True:
        chunk = fs.read(chunk_size)
        # 讀到檔尾無資料時會回傳空的 bytes 物件 (b'')，視為 False
        if not chunk:
            break
        # 將暫存資料寫入目的檔案
        fd.write(chunk)
```

### 2. 讀寫二進位檔 (簡易版)
如果檔案不大，可以使用更簡易的方法一次讀寫完畢。
```python
from pathlib import Path

source = Path("image.png")
destination = Path("binary.png")

# read_bytes() 和 write_bytes() 已內含 with 區塊，執行完會自動關閉檔案
data = source.read_bytes()
destination.write_bytes(data)
```

## 存取圖檔與影像處理 (Pillow 套件)
在 Python 中處理圖片，最常用的是第三方套件 `Pillow` (匯入時為 `PIL`)，它提供了豐富的影像讀寫與編輯功能。

### 1. 讀取與儲存圖檔
```python
from pathlib import Path
from PIL import Image

source = Path("image.png")
destination = Path("output.png")

# 開啟圖檔並取得相關資訊
with Image.open(source) as image:
    # 影像屬性可透過以下方式取得：
    # image.format (格式), image.size (尺寸), image.mode (色彩模式)
    
    # 顯示圖片 (會自動呼叫作業系統預設的圖片檢視器)
    image.show()
    
    # 將圖片另存新檔
    image.save(destination)
```

### 2. 影像編輯操作
`Pillow` 支援各種基礎影像處理，如縮放、旋轉、轉換色彩模式與裁切：
```python
from PIL import Image
from pathlib import Path

file = Path("image.png")

with Image.open(file) as image:
    # 縮放：調整影像大小為 200 x 200
    resized_image = image.resize((200, 200))

    # 旋轉：逆時針旋轉 45 度
    rotated_image = image.rotate(45)

    # 轉換色彩模式："L" 代表轉換為灰階 (Grayscale)
    gray_image = image.convert("L")

    # 裁切：輸入裁切區域的座標 (左, 上, 右, 下)
    cropped_image = image.crop((50, 50, 200, 200))
```
