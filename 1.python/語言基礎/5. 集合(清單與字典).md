## 一、 Tuple (元組)
- **特性**: 有序、**不可變 (Immutable)**、可重複。
- **語法**: `(value1, value2)`
- > [!TIP]
  > 常用於存放不希望被修改的資料，例如座標或設定值。

```python
# 一、 Tuple (元組) 範例
point = (10, 20)
print(f"X 座標: {point[0]}, Y 座標: {point[1]}") # 輸出: X 座標: 10, Y 座標: 20

# 元組拆包 (Unpacking)
x, y = point
print(f"解包後 -> x: {x}, y: {y}")

# 嘗試修改元組會報錯
# point[0] = 99  # TypeError: 'tuple' object does not support item assignment
```

## 二、 List (串列)
- **特性**: 有序、**可變 (Mutable)**、可重複。
- **常用操作**:
    - `append()` / `insert()`: 新增資料。
    - `remove()` / `pop()`: 刪除資料。
    - `sort()`: 排序 (會改變原始資料)。
- **加工處理**:
    - `split()`: 將字串切割成 List。
    - `map(function, iterable)`: 對序列中每個元素執行相同動作。

```python
# 二、 List (串列) 常用操作範例
fruits = ["apple", "banana"]
fruits.append("cherry")      # 附加至尾端 -> ['apple', 'banana', 'cherry']
fruits.insert(1, "orange")   # 插入至索引 1 -> ['apple', 'orange', 'banana', 'cherry']
print(fruits)

fruits.remove("banana")      # 移除指定元素 -> ['apple', 'orange', 'cherry']
popped_fruit = fruits.pop(0) # 移除並回傳索引 0 的元素 -> 移除 'apple'
print(f"被移除的: {popped_fruit}, 剩餘的: {fruits}") # 輸出: 被移除的: apple, 剩餘的: ['orange', 'cherry']

# 排序本體
nums = [3, 1, 4, 1, 5]
nums.sort()                  # 直接改變原串列排序
print(f"排序後: {nums}")      # 輸出: 排序後: [1, 1, 3, 4, 5]

# split 與 map 範例
text = "10,20,30"
str_list = text.split(",")   # 切割字串 -> ['10', '20', '30']
# 將清單內的所有字串元素轉為整數
int_list = list(map(int, str_list)) 
print(int_list)              # 輸出: [10, 20, 30]
```

### 搜尋功能
- `in` / `not in`: 檢查元素是否存在。
- **二元搜尋 (bisect)**: 需先排序，適用於大數據快速查找。

```python
# 搜尋與二元搜尋範例
names = ["Alice", "Bob", "Charlie"]
print("Bob" in names)    # 輸出: True

# bisect 二元搜尋 (必須先將清單排序)
import bisect
sorted_nums = [10, 20, 30, 40, 50]
# 尋找 25 應該插入在排序好的清單中的哪個索引位置，以維持排序
insert_pos = bisect.bisect(sorted_nums, 25)
print(f"25 應該插入在索引: {insert_pos}") # 輸出: 2 (即 20 與 30 之間)
```

### 常用函數
- filter(function, iterable) : 留下符合條件的值
- max(iterable, key=function ) :可以直接輸入iterable獲得該清單的最大值，可以添加key過濾條件
- min(iterable, key=function) : 可以直接輸入iterable獲得該清單的最小值，可以添加key過濾條件

```python
# List 常用內建函數範例
numbers = [1, -2, 3, -4, 5]

# filter: 過濾出大於 0 的正數
positives = list(filter(lambda x: x > 0, numbers))
print(f"正數清單: {positives}") # 輸出: [1, 3, 5]

# max / min 搭配 key 自定義過濾規則
# 尋找絕對值最大的數值
abs_max = max(numbers, key=abs)
print(f"絕對值最大值: {abs_max}") # 輸出: 5 (因為 abs(-4)=4, abs(5)=5)
```

## 三、 Dictionary (字典)
- **特性**: 鍵值對 (Key-Value)、Key 不可重複。
- **語法**: `{"key": value}`
- **方法**:
    - `.keys()`: 取得所有鍵。
    - `.values()`: 取得所有值。
    - `.items()`: 取得所有鍵值對。

```python
# 三、 Dictionary (字典) 範例
student = {"name": "Bob", "age": 20, "score": 95}

# 取得所有的 Keys, Values 與 Items
print(list(student.keys()))   # 輸出: ['name', 'age', 'score']
print(list(student.values())) # 輸出: ['Bob', 20, 95]
print(list(student.items()))  # 輸出: [('name', 'Bob'), ('age', 20), ('score', 95)]

# 使用 get() 安全取值 (避免 key 不存在時報錯)
email = student.get("email", "無提供電子郵件")
print(email) # 輸出: 無提供電子郵件

# 更新與新增鍵值對
student.update({"score": 100, "major": "CS"})
print(student) # 輸出: {'name': 'Bob', 'age': 20, 'score': 100, 'major': 'CS'}
```

## 四、 Set (集合)
- **特性**: **無序**、不可重複。
- **語法**: `set_name = set()`
- > [!WARNING]
  > **注意**：不能使用 `s = {}` 建立空集合，這會被辨識為字典。
- **元素刪除**:
    - `remove()`: 若元素不存在會報錯。
    - `discard()`: 若元素不存在 **不會報錯**。

### 數學運算
- `intersection()`: 交集。
- `union()`: 聯集。
- `difference()`: 差集。

```python
# 四、 Set (集合) 範例與數學運算
s = {1, 2, 3}
# s.remove(9)    # 錯誤！KeyError
s.discard(9)     # 安全刪除！即使 9 不在集合中，也不會發生任何錯誤

# 集合數學運算
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a.intersection(b)) # 輸出: {3, 4} (交集，同 a & b)
print(a.union(b))        # 輸出: {1, 2, 3, 4, 5, 6} (聯集，同 a | b)
print(a.difference(b))   # 輸出: {1, 2} (差集，同 a - b)
```
