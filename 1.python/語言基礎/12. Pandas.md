## 1. Pandas 概論
Pandas 是一個用於資料處理與分析的 Python 擴充套件，安裝名稱為 `pandas`。它衍生自計量經濟學術用語「panel data」。

Pandas 主要提供兩種資料結構：
- **Series**：類似 1 維陣列，帶有標籤索引 (index)。
- **DataFrame**：類似試算表表格，為 2 維資料結構。

---

## 2. Series 概論與操作
`Series` 是一維的資料結構，內容可以來自 `list`、`ndarray` 或 `dictionary`。我們也可以透過索引或自訂的 `index label` 來存取元素。

### 2-1 建立 Series
```python
import numpy as np
from pandas import Series

# --- 來自 list ---
# dtype="string" 可指定資料型別
s_list = Series(["Python", "Java", "JS"], dtype="string")

# --- 來自 ndarray ---
array = np.array(["Python", "Java", "JS"])
s_array = Series(array)

# --- 來自 dictionary ---
# key 會自動成為 index label
s_dict = Series({
    "name": "Python",
    "price": 500,
    "author": "Paul"
})
```

### 2-2 Series 基本資訊與轉換
```python
from pandas import Series

series = Series({"name": "Python", "price": 500})

# --- 取得資訊 ---
series.dtype          # 取得型別
series.shape          # 取得形狀 (例如: (2,))
series.memory_usage() # 取得記憶體使用量 (byte)
series.count()        # 取得非空值的元素數量

# --- 型別轉換 ---
series.tolist()       # 轉為 list
series.to_numpy()     # 轉為 ndarray
series.to_dict()      # 轉為 dictionary
```

### 2-3 增刪改查與走訪
```python
from pandas import Series

series = Series(["Python", 500, "Paul"])

# --- 讀取 (查) ---
series[0]             # 透過位置索引取值: "Python"
Series(series, index=[0, 1]) # 取出局部 Series

# 設定自訂 index label
series.index = ["name", "price", "author"]
series["name"]        # 透過標籤取值: "Python"

# --- 走訪 ---
for value in series:
    pass # 走訪每個值
for index, value in series.items():
    pass # 走訪每個標籤與值

# --- 新增 / 修改 ---
series["isbn"] = "123456"   # 新增
series["author"] = "Peter"  # 修改已存在的屬性

# --- 刪除 ---
# drop 會回傳一個「新的 Series」，不會修改原始 series
new_series = series.drop("author")
```

## 3. DataFrame 概論與建立
`DataFrame` 類似表格，內容可以來自 2 維 `list`、`ndarray`、`dictionary` 或自訂物件的集合。

### 3-1 建立 DataFrame
```python
import numpy as np
from pandas import DataFrame

# --- 來自 2維 list ---
# 內部 list 長度可以不同，缺值的部分會自動補 NaN
df_list2D = DataFrame([
    ["Python", 500],
    ["Java", 400],
    ["Swift"]
])
# 加上自訂欄位名稱 (Column Labels)
df_list2D.columns = ["name", "price"]

# --- 來自 2維 ndarray ---
# 注意：ndarray 各列長度必須相同，否則會產生 ValueError
array2D = np.array([["Python", 500], ["Java", 400]])
df_array2D = DataFrame(array2D)

# --- 來自 Dict-List (字典中包含 List) ---
# key 為欄位名稱，list 長度必須相同
df_dict_list = DataFrame({
    "name": ["Python", "Java", "Swift"],
    "price": [500, 450, 550]
})

# --- 來自 List-Dict (List 中包含字典) ---
# list 中每個字典代表一列，長度可以不同，缺值補 NaN
df_list_dict = DataFrame([
    {"name": "Python", "price": 500},
    {"name": "Java", "price": 450},
    {"name": "Swift"}
])
# 可將某個欄位設定為列索引 (Row Index)
df_list_dict.set_index("name", inplace=True)
```

### 3-2 DataFrame 資訊與轉換
```python
from pandas import DataFrame

df = DataFrame({
    "name": ["Python", "Java"],
    "price": [500, 450]
})

# --- 取得資訊 ---
df.info()     # 顯示欄位、資料型別與記憶體用量等完整資訊
df.index      # 取得列索引 (Row Index)
df.columns    # 取得欄位名稱 (Column Labels)

# --- 型別轉換 ---
df.to_dict(orient="list")    # 轉成 Dictionary
df.to_numpy()                # 轉成 ndarray
df.to_numpy().tolist()       # 轉成 List (需先轉為 ndarray)
# df.to_dict(orient="records") 可將每一列轉成獨立字典的 List，適合用來還原成自訂物件
```

```python
import pandas as pd
import matplotlib.pyplot as plt

df = pd.DataFrame({
    "months": ["1", "2", "3"],
    "Units": [1000, 1200, 1350],      # 銷售數量
    "Profit": [100000, 120000, 135000] # 銷售利潤

ax2 = ax1.twinx()

# 將兩個坐標軸的圖例合併
handles1, labels1 = ax1.get_legend_handles_labels()
handles2, labels2 = ax2.get_legend_handles_labels()
ax1.legend(handles1 + handles2, labels1 + labels2, loc="best")

plt.show()
```

## 6. 資料處理 (增刪改查)
DataFrame 提供了靈活的方法進行資料過濾、新增、修改與刪除。

### 6-1 讀取與查詢
```python
from pandas import DataFrame

df = DataFrame({
    'name': ['John', 'Mary', 'Ken'],
    'age': [30, 25, 35],
    'city': ['Taipei', 'Taichung', 'Kaohsiung']
})

# --- 選取欄位 ---
df['name']           # 取得單一欄位 (回傳 Series)
df[['name', 'age']]  # 取得多個欄位 (回傳 DataFrame)

# --- 透過 loc 取得列資料 ---
df.loc[0]            # 取得列索引為 0 的資料
df.loc[0:2]          # 取得列索引 0 到 2 (包含 2，與一般 list 切片不同)
df.loc[[0, 1]]       # 取特定的多列
df.loc[0, 'name']    # 取列索引 0 且欄位為 'name' 的值

# --- 條件查詢 ---
df[df['age'] >= 30]  # 取 age 大於等於 30 的資料列
# 複合條件: 使用 & (and), | (or), ~ (not)，條件需用小括號包覆
df[(df['age'] >= 30) & (df['city'] == 'Taipei')]

# --- 取前後幾筆 ---
df.head(2) # 讀取前 2 筆
df.tail(2) # 讀取後 2 筆
```

### 6-2 新增與修改
```python
from pandas import DataFrame

df = DataFrame(columns=['name', 'age'])

# --- 新增 ---
# 透過 loc 依序新增列資料
df.loc[0] = ['John', 30]
df.loc[1] = ['Mary', 25]

# 新增一整欄
df['city'] = ['Taipei', 'Taichung']

# --- 修改 ---
# 將名叫 John 的年齡改為 33
df.loc[df['name'] == 'John', 'age'] = 33
```

### 6-3 刪除資料
```python
from pandas import DataFrame

df = DataFrame({
    'name': ['John', 'Mary', 'Ken'],
    'age': [30, 25, 35],
    'city': ['Taipei', 'Taichung', 'Kaohsiung']
})

# --- 刪除列 ---
# 1. 條件查詢法 (推薦): 取得除了 John 以外的資料
df_deleted = df[df['name'] != 'John']

# 2. 使用 drop() 方法
df.drop(df[df['name'] == 'John'].index)

# --- 刪除欄位 ---
df.drop(columns='city')               # 刪除單一欄
df.drop(columns=['age', 'city'])      # 刪除多欄
# df.drop(columns=df.columns) 刪除所有欄
# df.drop(index=df.index) 刪除所有列
```

## 7. 資料重塑
資料重塑 (Data Reshaping) 是指對資料的結構進行重新組織，常見的方法包含合併 (Merge/Join)、轉置 (Transpose) 以及長寬格式轉換 (Melt/Pivot)。

### 7-1 合併 (Merge & Join)
- `merge()`：基於欄位 (Column) 合併，用途廣泛。
- `join()`：基於索引 (Index) 合併。

合併方式 (`how` 參數)：
- `inner` (預設)：只保留兩邊都有對應值的資料。
- `left`：保留左邊所有資料，右邊沒有匹配的填 `NaN`。
- `right`：保留右邊所有資料，左邊沒有匹配的填 `NaN`。
- `outer`：保留所有資料，沒有匹配的填 `NaN`。

```python
from pandas import DataFrame

# --- 1. merge (依據特定欄位) ---
df1 = DataFrame({"Store_ID": ["S001", "S002"], "Name": ["Taipei", "Taichung"]})
df2 = DataFrame({"ID": ["S001", "S002"], "Revenue": [850, 730]})

# 欄位名稱不同時，使用 left_on 與 right_on 指定
result_merge = df1.merge(df2, left_on="Store_ID", right_on="ID", how="inner")

# --- 2. join (依據 Index) ---
df3 = DataFrame({"Name": ["Taipei", "Taichung"]}, index=["S001", "S002"])
df4 = DataFrame({"Revenue": [850, 730]}, index=["S001", "S002"])

# 依據列索引合併
result_join = df3.join(df4, how="left")
```

### 7-2 轉置 (Transpose)
將列與行互換。
```python
from pandas import DataFrame

df = DataFrame({"A": [1, 2], "B": [3, 4]})

# 下列兩種寫法結果相同
df_transposed = df.T
df_transposed = df.transpose()
```

### 7-3 長寬格式轉換 (Long vs. Wide format)
- `melt()`：將寬格式 (多個欄位攤平成一列一觀測值) 轉成長格式。適用於機器學習或繪圖前的資料整理。
- `pivot()`：將長格式轉回寬格式。適用於增加人類閱讀的便利性。

```python
from pandas import DataFrame

df_wide = DataFrame({
    "months": [1, 2],
    "A": [1000, 1200],
    "B": [300, 320]
})

# --- 寬轉長 (melt) ---
df_long = df_wide.melt(
    id_vars="months",        # 保留不被拆開的識別欄位
    value_vars=["A", "B"],   # 預設為除了 id_vars 外的所有欄位
    var_name="products",     # 原先的欄位名稱 (A, B) 將被存入的新欄位
    value_name="sales"       # 原先的值將被存入的新欄位
)

# --- 長轉寬 (pivot) ---
df_pivoted = df_long.pivot(
    index="months",
    columns="products",
    values="sales"
).reset_index() # 還原 months 為普通欄位
```

## 9. 資料清理
資料清理包含缺失值處理、格式轉換、重複值移除與錯誤值過濾。

### 9-1 缺失值處理 (NaN)
```python
import pandas as pd
import numpy as np

# pandas 讀入 csv 時，若有缺失值預設會轉成 NaN
df = pd.read_csv("raw.csv", skipinitialspace=True)

# 1. 檢查空值
df.isna()

# 2. 移除空值 (整列移除)
df.dropna()
df.dropna(subset=['price']) # 只針對特定欄位判斷空值

# 3. 填補空值
df["price"] = df["price"].fillna(0) # 以 0 填補空值

# 4. 自訂空值並替換
# 在讀取時直接指定：pd.read_csv("raw.csv", na_values=["na", "--"])
# 或事後替換：
df = df.replace(["na", "--"], np.nan)
```

### 9-2 格式轉換 (日期時間)
```python
import pandas as pd

df = pd.read_csv("raw.csv")

# to_datetime 常用參數：
# format="ISO8601" (速度快), format="%Y/%m/%d" (速度最快), format="mixed" (速度慢)
# errors="coerce" 遇到錯誤格式時轉成 NaT (時間空值)，而不是丟出執行錯誤
df["date_parsed"] = pd.to_datetime(df["date"], format="ISO8601", errors="coerce")

# 轉換失敗變成 NaT 的資料，一樣可用 dropna 移除
df.dropna(subset=['date_parsed'])
```

### 9-3 移除重複與錯誤資料
```python
import pandas as pd

df = pd.read_csv("raw.csv")

# --- 移除重複資料 ---
df.duplicated(keep="first") # 檢查重複，預設保留第一筆
df.drop_duplicates()        # 移除所有欄位皆相同的重複列
df.drop_duplicates(subset=["name"]) # 只判斷特定欄位是否重複

# --- 移除錯誤資料 (條件過濾) ---
# 例如：移出價格小於 0 的錯誤資料
df_cleaned = df[~(df["price"] < 0)]
```

## 10. 數學計算與排序
Pandas 一樣支援廣播機制的數學運算，並提供方便的排序功能。

### 10-1 基本運算與排序
```python
from pandas import DataFrame, Series

df1 = DataFrame([[7, 8, 9], [10, 11, 12]], columns=["A", "B", "C"])
s1 = Series([1, 2, 3])

# --- 數學運算 ---
# 2維 (DataFrame) 可以跟 1維 (Series) 運算，但對應的 shape 要相容
df1 + s1
df1 - s1
df1 * s1
df1 / s1

# --- 排序 ---
df = DataFrame([
    [3, 11, 9],
    [5, 2, 15],
    [1, 6, 99]
], columns=["A", "B", "C"])

# 依據單一欄位排序 (預設升冪)
df.sort_values(by="A", axis=0)

# 將每一列的元素各自水平排序
df.apply(sorted, axis=1)
```
