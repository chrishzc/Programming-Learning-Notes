# 6. 函式 (Functions)

> [!ABSTRACT]
> 學習如何定義與呼叫函式，掌握參數傳遞（預設、位置、關鍵字）、變數作用域、遞迴函式與匿名函數 (Lambda)。

---

函式是自成一個的程式區塊 (block)，用於執行特定功能或特定計算。

#### 一個函式通常包含：
- **名稱**: 方便之後重複呼叫執行。
- **參數 (Parameter)**: 用於接收外部傳入的資料。
- **回傳值 (Return Value)**: 運算後的結果，可傳回呼叫處。

```python
def function_name(parameter):
    # 功能內容
    return result
```

```python
# 基礎函式定義與呼叫範例
def greet(user):
    return f"哈囉，{user}！歡迎學習 Python。"

# 呼叫函式並印出回傳值
message = greet("Alice")
print(message) # 輸出: 哈囉，Alice！歡迎學習 Python。
```
函式也可以:
#### 指派
像變數指派⼀樣，將⼀個函式指派給其他函式 ✦ op_func = divide 
#### 指派後呼叫
呼叫被指派的函式就等於呼叫原來的函式 ✦ op_func(10, 2)
#### 被回傳
可以在運算結果回傳函數當作結果

```python
# 函式作為一等公民 (First-Class Citizens) 範例
def add(x, y):
    return x + y

# 1. 函式指派給變數
calc = add
print(calc(5, 3)) # 輸出: 8

# 2. 函式作為回傳值 (高階函式)
def get_operator(op_type):
    def multiply(x, y):
        return x * y
    if op_type == "mul":
        return multiply # 回傳函式本身

my_func = get_operator("mul")
print(my_func(4, 5)) # 輸出: 20
```

### 1. 參數 (Parameter)
- 可以設定多個參數，中間用逗號 `,` 隔開。
- **預設值 (Default Argument)**: 
    - 呼叫時若不傳值，則使用預設值。
    - > [!IMPORTANT]
    - > 如果一個參數設有預設值，其後的「所有參數」也都必須設有預設值。

```python
# 預設參數範例
def create_profile(name, age=18, city="Taipei"):
    return f"姓名: {name}, 年齡: {age}, 城市: {city}"

# 正常使用預設參數
print(create_profile("Alice"))             # 輸出: 姓名: Alice, 年齡: 18, 城市: Taipei
print(create_profile("Bob", age=25))       # 輸出: 姓名: Bob, 年齡: 25, 城市: Taipei

# 錯誤示範 (含有預設值的參數後面不能有一般參數)
# def wrong_func(age=18, name): # SyntaxError: non-default argument follows default argument
#     pass
```

### 2. 回傳值 (Return Value)
- 使用 `return` 將結果傳回。
- 若函式內沒有 `return`，預設會回傳 `None`。
- > [!TIP]
    > **多重回傳**：Python 支援回傳多個值，本質上會封裝成一個 **Tuple**。
    > ```python
    > def get_data():
    >     return 1, 2  # 回傳 (1, 2)
    > ```

### 3. 引數 (Arguments)
- 呼叫函式時實際傳入的值稱為「引數」。
- 位置引數(atg): 依照順序對應參數。
- 關鍵字引數(kwargs): 指定 `參數名=值`，順序可調換。
### 3.1 引數打包與拆解
函式呼叫⽀援 list / tuple 與 dictionary 引數⾃動打包 (packing) 與拆解 (unpacking)
- list / tuple：使⽤「\*」
- dictionary：使⽤「\*\*」

```python
# 位置與關鍵字引數、打包與拆解範例
# 1. 定義接收任意個數引數的函式
def show_info(*args, **kwargs):
    # args 會被打包成一個 Tuple -> (1, 2, 3)
    # kwargs 會被打包成一個 Dict -> {'a': 'Apple', 'b': 'Banana'}
    print("args:", args)
    print("kwargs:", kwargs)

show_info(1, 2, 3, a="Apple", b="Banana")

# 2. 將現有容器拆解 (Unpacking) 後傳入函式
def multiply(x, y, z):
    return x * y * z

numbers_list = [2, 3, 4]
# 使用 * 拆解 list 為位置引數傳入
print(multiply(*numbers_list)) # 輸出: 24 (相當於 multiply(2, 3, 4))

dict_params = {"x": 2, "y": 3, "z": 5}
# 使用 ** 拆解 dict 為關鍵字引數傳入
print(multiply(**dict_params)) # 輸出: 30 (相當於 multiply(x=2, y=3, z=5))
```

## 4. 遞迴函式
- 函式本身會再次呼叫自己，但要注意加上限制避免變成無限迴圈。
- 遞迴函式功能通常也可使遊迴圈功能取代

```python
# 遞迴函式範例：計算階乘 (n!)
# 階乘定義：n! = n * (n-1) * (n-2) * ... * 1
def factorial(n):
    # 終止條件 (Base Case)
    if n == 1:
        return 1
    # 遞迴呼叫 (Recursive Case)
    return n * factorial(n - 1)

print(factorial(5)) # 輸出: 120 (5 * 4 * 3 * 2 * 1)
```

## Lambda
lambda 是⼀個沒有名字、極其簡化的函式
- 必須是運算式，所以無需加上return 
- 可以將 lambda 指派給變數
- 常⽤於函式型參數的傳遞

```python
# Lambda 匿名函式範例
# 1. 簡單的 lambda 運算
square = lambda x: x ** 2
print(square(4)) # 輸出: 16

# 2. 常用於 sorted 排序條件 (例如依據 Tuple 的第二個元素排序)
pairs = [(1, 'one'), (2, 'two'), (3, 'three'), (4, 'four')]
# 依字串長度排序
pairs_sorted = sorted(pairs, key=lambda pair: len(pair[1]))
print(pairs_sorted) # 輸出: [(1, 'one'), (2, 'two'), (4, 'four'), (3, 'three')]
```
