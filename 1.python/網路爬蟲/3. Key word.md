# 3. 網路爬蟲關鍵字與 HTTP 協定

> [!ABSTRACT]
> 本章介紹 HTTP 協定中的核心概念，包含 Request Headers、Cookies、Payload 以及 HTTP 請求方法，協助開發者突破網站的反爬蟲限制。

---

## 一、 HTTP 請求核心要素

當爬蟲模擬瀏覽器向伺服器發送請求時，主要會帶有以下幾個重要資訊：

### 1. 請求標頭 (Request Headers)
標頭是用於提供客戶端環境資訊的元數據。許多網站會檢查標頭來過濾機器人。
- **User-Agent**：聲明客戶端的操作系統、瀏覽器類型與版本。這是爬蟲必設的標頭，否則預設會顯示 `python-requests`，極易被伺服器阻擋。
- **Referer**：表示當前請求是由哪一個來源網頁連結過來的。部分圖片或 API 會有防盜鏈機制，要求必須帶有特定的 Referer。

### 2. Cookie (餅乾/緩存憑證)
Cookie 是伺服器存在客戶端瀏覽器的小型文字檔，用於記錄使用者的登入狀態、購物車內容或點擊行為。
- **爬蟲應用**：當需要爬取必須登入才能看到的網頁（如 PTT 實業坊的八卦版確認滿 18 歲，或是會員專區）時，必須在請求中帶入對應的 Cookie 值。

### 3. 資料載體 (Payload)
發送請求時傳遞給伺服器的資料參數：
- **查詢參數 (Query String Parameters)**：通常用於 `GET` 請求，直接附加在網址後方（例如 `url?key1=value1&key2=value2`）。
- **表單數據 (Form Data) 或 JSON 載體**：通常用於 `POST` 請求，將資料放在 HTTP Body 中傳送（如登入時的帳號密碼）。

---

## 二、 URL 與 URI 的區別

```
                    URI (Uniform Resource Identifier)
 ┌──────────────────────────────────┴──────────────────────────────────┐
 https://www.example.com/forum/questions.html?tag=python#top
 └──────────┬──────────┘└─────────┬─────────┘└────┬────┘└──┬──┘
         Protocol             Path            Query     Fragment
 └──────────────────────┬───────────────────────┘
                       URL (Uniform Resource Locator)
```

- **URI (Uniform Resource Identifier，統一資源識別碼)**：是一個用於識別某個網際網路資源的字串標識。
- **URL (Uniform Resource Locator，統一資源定位器)**：是 URI 的子集，除了識別資源外，還提供了如何定位（找到）該資源的路徑（如 `https://...`）。

---

## 三、 HTTP 請求方法

最常用的 HTTP 請求方法包括：

- **`GET`**：向伺服器索取資料。參數會直接暴露在 URL 中，傳輸資料量有限，且會被瀏覽器記錄。
- **`POST`**：向伺服器提交新資料。資料存放在 Request Body 中，不會顯示在網址列上，安全性較高，適合用來進行登入或提交表單。

## 四、 實戰演練：在 Requests 中自訂 Headers 與 Cookies

```python
import requests

url = "https://www.ptt.cc/bbs/Gossiping/index.html"

# 1. 自訂 Headers 模擬 Chrome 瀏覽器，避免被判定為 Python 腳本
headers = {
    "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36"
}

# 2. 設定 Cookies 以繞過 PTT 八卦版的 18 歲年齡確認頁面
cookies = {
    "over18": "1"
}

# 發送帶有 headers 與 cookies 的 GET 請求
response = requests.get(url, headers=headers, cookies=cookies)

if response.status_code == 200:
    print("成功繞過年齡驗證！網頁標題片段：")
    print(response.text[:200]) # 印出網頁前 200 個字元
else:
    print(f"請求失敗，狀態碼: {response.status_code}")
```
