# 6. 動態網頁爬取 (Selenium & API 偵測)

> [!ABSTRACT]
> 本章介紹如何爬取採用 JavaScript 非同步加載或 AJAX 技術的動態網頁。內容涵蓋靜態與動態網頁的本質區別、如何使用瀏覽器開發者工具進行網路（Fetch/XHR）逆向偵測以直接抓取後台 API，以及使用瀏覽器自動化測試工具 Selenium 模擬真人點擊與動態滾動的實戰程式碼。

---

## 一、 靜態網頁與動態網頁的差異

傳統爬蟲與現代爬蟲在抓取策略上有著根本的差別，主要源於網頁渲染機制的轉變：

- **靜態網頁 (Static Page)**：
  - **特徵**：伺服器回傳的 HTML 原始碼中就已經包含了所有網頁內容。
  - **爬取方式**：直接使用 `requests.get()` 下載網頁，再用 `BeautifulSoup` 即可輕鬆解析。
- **動態網頁 (Dynamic Page / Single Page Application)**：
  - **特徵**：伺服器回傳的 HTML 幾乎是個空殼。網頁載入後，瀏覽器會執行 JavaScript 腳本，透過非同步請求（AJAX / Fetch）向伺服器索取 JSON 資料，再動態渲染到網頁上。
  - **爬取困境**：如果直接使用 `requests.get()`，只會抓到未經渲染的空白網頁（查無任何數據）。

---

## 二、 網頁逆向工程：XHR / API 請求偵測

這是爬取動態網頁**最推薦、最高效且速度最快**的方案。我們不需要去開瀏覽器，而是直接去抓網頁背後的資料庫 API。

### 步驟流程：
1. 使用 Chrome 或 Edge 瀏覽器打開目標網站。
2. 按下 `F12` 鍵開啟開發者工具，切換至 **`Network`（網路）** 面板。
3. 點選 **`Fetch/XHR`** 標籤，然後重新整理網頁（或在網頁點選「下一頁」、「加載更多」）。
4. 在左側出現的請求列表中尋找，點擊 `Preview` 預覽，直到發現回傳結構化 JSON 資料的請求。
5. 切換至 `Headers`，複製其 **Request URL**。
6. 使用 `requests` 直接對該 URL 發送請求，並使用 `.json()` 讀取結果：

```python
import requests

api_url = "https://example.com/api/get_products?page=1"
headers = {"User-Agent": "Mozilla/5.0"}

response = requests.get(api_url, headers=headers)
data = response.json() # 將回傳的 JSON 直接轉成 Python 字典

for product in data["items"]:
    print(f"商品: {product['title']}, 價格: {product['price']}")
```

## 三、 瀏覽器自動化工具：Selenium

當後台的 API 設有複雜的加密特徵碼（Signature / Tokens）導致逆向分析極度困難，或者網頁操作極其繁瑣（需要複雜的滑動滑鼠、拖曳拼圖驗證等），我們就必須使用 **Selenium**。

- **運作原理**：
  - Selenium 通過瀏覽器驅動程式（WebDriver）直接驅動真實的瀏覽器（如 Chrome, Firefox）執行操作。瀏覽器會自動下載、執行 JS 並完成網頁渲染，我們只需像操作本機軟體一樣去讀取網頁 DOM 樹。

### 核心操作方法：
- **等待機制**：動態網頁物件需要時間加載，必須使用**顯式等待 (Explicit Wait)**，否則會因為找不到元素而報錯。
```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

driver = webdriver.Chrome()
driver.get("https://example.com")

# 顯式等待：最多等待 10 秒，直到 class 為 'product-list' 的元素出現
element = WebDriverWait(driver, 10).until(
    EC.presence_of_element_located((By.CLASS_NAME, "product-list"))
)
```

## 四、 實戰演練：Selenium 動態滾動網頁爬取

以下程式碼示範如何使用 Selenium 自動開啟瀏覽器、模擬滾動滑鼠至頁面最底端以加載更多動態資料，並提取商品名稱：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
import time

# 1. 啟動 Chrome 瀏覽器 (可加入無頭模式參數以隱藏視窗)
options = webdriver.ChromeOptions()
# options.add_argument('--headless') # 啟用無頭模式 (不彈出瀏覽器視窗)
driver = webdriver.Chrome(options=options)

try:
    # 2. 前往目標動態載入網頁
    driver.get("https://example.com/infinite-scroll-gallery")
    
    # 3. 模擬下拉滾動 3 次以加載更多商品
    for i in range(3):
        # 執行 JavaScript 捲動到網頁最下方
        driver.execute_script("window.scrollTo(0, document.body.scrollHeight);")
        time.sleep(2) # 等待 2 秒讓資料下載與渲染
        
    # 4. 等待最後一批商品圖片加載完成
    WebDriverWait(driver, 10).until(
        EC.presence_of_element_located((By.CLASS_NAME, "gallery-item"))
    )
    
    # 5. 提取網頁渲染完成後的元素
    items = driver.find_elements(By.CLASS_NAME, "item-title")
    for index, item in enumerate(items):
        print(f"[{index+1}] 商品名稱: {item.text}")
        
finally:
    # 6. 關閉瀏覽器，釋放資源
    driver.quit()
```
