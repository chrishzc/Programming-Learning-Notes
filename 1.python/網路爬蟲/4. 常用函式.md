# 4.網路爬蟲常用函式與套件

> [!ABSTRACT]
> 介紹如何使用 Requests 取得網頁原始碼、BeautifulSoup 解析網頁結構，以及 Python 內建的檔案讀寫操作。

---

## 1. 取得網頁原始碼 (Requests 套件)
使用 `requests` 套件來發送 HTTP 請求並取得網頁內容。

```python
import requests

# 發送 GET 請求
response = requests.get("https://example.com")

# 常用屬性與方法
print(response.status_code) # 取得狀態碼 (例如 200 代表成功, 404 找不到網頁)
print(response.text)        # 取得網頁的 HTML 原始碼 (字串格式)
print(response.json())      # 將回傳的 JSON 格式資料轉為 Python 字典 (用於 API 爬蟲)
```

## 2. 解析網頁結構 (BeautifulSoup4 尋找功能)
使用 `bs4` (BeautifulSoup) 將 HTML 字串轉換為可以快速搜尋的結構。

```python
from bs4 import BeautifulSoup

soup = BeautifulSoup(response.text, "html.parser")

# --- 常用尋找方法 ---

# .find()：只能找單層標籤，尋找第一個符合條件的標籤。找不到回傳 None。
title_tag = soup.find('h1') 
print(title_tag.text) # 取得標籤內的文字內容

# .find_all()：找所有符合條件的標籤，並輸出成 list (串列)。
links = soup.find_all('a')
for link in links:
    print(link.get('href')) # 取得標籤的屬性值 (例如 href 連結)

# .select()：使用 CSS 選擇器尋找，可以找多層或特定 class/id，回傳 list。
items = soup.select('div.content > p.text') # 尋找 class 為 content 的 div 底下，class 為 text 的 p 標籤

# .select_one()：同上，但只回傳第一個找到的結果。
first_item = soup.select_one('#main-id') # 尋找 id 為 main-id 的標籤
```

## 3. 檔案處理 (內建函式與檔案存取)
爬蟲抓取資料後，通常需要將資料儲存到本機的檔案中（例如 txt, csv 等格式）。

```python
# --- 內建檔案操作 ---

# 傳統寫法：開啟檔案 -> 寫入/讀取 -> 關閉檔案
# mode="w" (寫入/覆寫), "a" (附加), "r" (讀取)
file = open("data.txt", mode="w", encoding="utf-8")
file.write("爬蟲取得的第一行資料\n") # 寫入單行文字
file.writelines(["第二行資料\n", "第三行資料\n"]) # 寫入多行 (串列形式)
file.close() # 記得關閉檔案以釋放系統資源

# 推薦寫法：使用 with open (離開區塊時會自動關閉檔案)
with open("data.txt", mode="r", encoding="utf-8") as f:
    content = f.read()        # 一次讀取全部內容成一個大字串
    print("全部內容：\n", content)
    
    # line = f.readline()     # 每次只讀取一行字串
    # lines = f.readlines()   # 讀取所有行，並回傳一個包含每一行的 list
```

## 4. RSS 訂閱源與 feedparser 套件

當網站提供 RSS (Really Simple Syndication) 或 Atom 訂閱源時，資料會以結構化的 XML 格式發佈。相較於解析混亂的 HTML 網頁，直接爬取 RSS 資訊更加穩定、乾淨。

**`feedparser`** 是 Python 中最受歡迎的 RSS/Atom 解析套件，能將複雜的 XML 結構自動轉換為直覺的 Python 字典 (Dict) 格式。

> [!NOTE]
> 使用前須使用 pip 安裝：`pip install feedparser`

### 💻 feedparser 範例程式碼

```python
import feedparser

# 1. 解析指定網址的 RSS Feed (以科技新聞訂閱源為例)
feed_url = "https://tw.news.yahoo.com/rss/technology"
feed = feedparser.parse(feed_url)

# 2. 取得訂閱源的基本資訊
print("頻道標題:", feed.feed.title)
print("頻道描述:", feed.feed.description)
print("-" * 50)

# 3. 遍歷並讀取所有文章項目 (預設取得 entries 串列)
for entry in feed.entries:
    # 常用欄位：title (標題)、link (連結)、published (發布日期)、summary (摘要)
    print(f"文章標題: {entry.title}")
    print(f"文章連結: {entry.link}")
    print(f"發布時間: {entry.published}")
    print(f"摘要內容: {entry.summary}\n")
```

> [!TIP]
> **資料清洗提示**
> RSS 提供的摘要（`entry.summary`）有時會包含 HTML 標籤（如 `<p>`, `<br>` 等）。在將資料存入資料庫前，可搭配 `BeautifulSoup(entry.summary, "html.parser").text` 來快速過濾掉所有 HTML 標籤，只保留乾淨的純文字。
