# 2. HTML 網頁結構基礎

> [!ABSTRACT]
> 本章介紹網頁的核心語言 HTML，了解網頁的基本組成結構、標籤與屬性表示法，並掌握常見的 HTML 元素，為網頁內容解析奠定基礎。

---

## 一、 什麼是 HTML？
HTML（HyperText Markup Language，超文字標記語言）是建構網頁結構的標準標記語言。它使用各種**標籤 (Tags)** 來標註網頁中的不同段落、標題、超連結、圖片等內容，由瀏覽器解析後呈現給使用者。

> [!TIP]
> **快速查看網頁原始碼**
> 在瀏覽器的網址列最前方加上 `view-source:`（例如 `view-source:https://example.com`），即可直接查看該網頁未經 JavaScript 渲染的完整 HTML 原始碼，這與爬蟲接收到的內容完全一致。

---

## 二、 標籤與屬性結構

HTML 的基本元素結構如下：

```
<tag_name attribute_name="attribute_value"> 網頁顯示內容 </tag_name>
```

- **起始標籤 (Start Tag)**：如 `<p>`，代表段落的開始。
- **屬性 (Attribute)**：寫在起始標籤內，提供額外資訊，以 `鍵="值"` 形式呈現（如 `href="https://google.com"`）。
- **內容 (Content)**：夾在起始與結束標籤之間的文字或圖片。
- **結束標籤 (End Tag)**：如 `</p>`，以斜線開頭，代表段落的結束。部分標籤（如 `<img>`、`<br>`）為單一標籤，不需要成對。

---

## 三、 HTML 基本結構範例

一個標準 of standard HTML 網頁結構如下：

```html
<!DOCTYPE html>
<html>
<head>    
    <meta charset="UTF-8">
    <title>我的第一個網頁</title>
</head>
<body>    
    <h1>Hello World!</h1>
    <p>這是一個網頁段落。</p>
    <a href="https://example.com">點擊這裡</a>
</body>
</html>
```

- **`<!DOCTYPE html>`**：宣告此文件為 HTML5 格式。
- **`<html>`**：根標籤，包覆整個網頁。
- **`<head>`**：網頁的標頭資訊（如字元編碼、網頁標題），此處內容不會直接顯示在網頁畫面上。
- **`<body>`**：網頁的實體內容，**這也是網蟲與爬蟲主要提取資料的區域**。

---

## 四、 爬蟲最常關注的 HTML 標籤列表

| 標籤元素 | 用途說明 | 範例與爬取重點 |
| :--- | :--- | :--- |
| **`<h1>` ~ `<h6>`** | 標題大小（`h1` 最大，`h6` 最小） | 用於抓取文章、新聞的標題。 |
| **`<p>`** | 段落文字 (Paragraph) | 用於擷取文章內文。 |
| **`<a>`** | 超連結 (Anchor) | 必須擷取其 **`href`** 屬性以獲得目標網址。 |
| **`<img>`** | 圖片 (Image) | 必須擷取其 **`src`** 屬性以獲得圖片下載連結。 |
| **`<div>`** | 區塊容器 (Block Container) | 最常見的網頁排版外框，用於限定爬取範圍。 |
| **`<span>`** | 行內容器 (Inline Container) | 用於包覆局部文字，常帶有特定的 class。 |
| **`<table>`** | 表格 (Table) | 包含 `<tr>` (列)、`<td>` (欄)，常存放排名、股價等結構化數據。 |
| **`<ul>` / `<ol>` / `<li>`** | 清單 (List) | 用於商品列表、文章清單等排版。 |
