# 8. 網頁定位解析速查 (CSS Selector & XPath)

> [!ABSTRACT]
> 本章為網頁定位語法速查指引。在進行 HTML 解析與爬蟲定位時，CSS 選擇器（CSS Selector）與 XML 路徑語言（XPath）是兩大核心定位語法。本章將提供兩者的詳細語法對照表，並提供在 BeautifulSoup 中使用 CSS 選擇器，以及在 lxml 中使用 XPath 提取資料的 Python 實戰程式碼。

---

## 一、 網頁定位解析簡介

在爬蟲程式下載網頁 HTML 後，必須精確定位到特定的標籤節點以提取資料。
- **CSS 選擇器 (CSS Selector)**：
  - 語法源於網頁樣式表（CSS），寫法簡潔直觀。**在 BeautifulSoup 中呼叫非常便利。**
- **XPath (XML Path Language)**：
  - 它是專門在 XML/HTML 文件中搜尋節點的語言。功能極其強大，支援**向上查找父節點**、**模糊文字比對**與**邏輯判斷**。常與 `lxml` 庫或 `Scrapy` 框架搭配。

---

## 二、 CSS 選擇器與 XPath 語法對照表

| 定位目的 | CSS 選擇器語法 | XPath 語法 |
| :--- | :--- | :--- |
| **根據標籤名定位** | `div` | `//div` |
| **根據 ID 定位** | `#content` | `//div[@id='content']` |
| **根據 Class 定位** | `.title` | `//div[contains(@class, 'title')]` |
| **根據特定屬性定位** | `a[href="link"]` | `//a[@href='link']` |
| **巢狀子節點 (直接子代)** | `ul > li` | `//ul/li` |
| **巢狀後代節點 (所有子代)** | `div li` | `//div//li` |
| **根據文字內容定位** | *無直接對應* | `//a[text()='下一頁']` |
| **取得節點的屬性值** | *需透過 BeautifulSoup 取得* | `//a/@href` |
| **取得節點的純文字** | *需透過 BeautifulSoup 取得* | `//div/text()` |

---

## 三、 BeautifulSoup 中的 CSS 選擇器實戰

BeautifulSoup 提供了兩個極為方便的 CSS 選擇器 API：
- **`select_one(selector)`**：回傳匹配到的第一個標籤物件（相當於 `find()`）。
- **`select(selector)`**：以清單回傳所有匹配到的標籤物件（相當於 `find_all()`）。

```python
from bs4 import BeautifulSoup

html_doc = """
<html>
  <body>
    <div id="main">
      <h1 class="title">今日新聞</h1>
      <ul class="news-list">
        <li><a href="link1.html">新聞一</a></li>
        <li><a href="link2.html">新聞二</a></li>
      </ul>
    </div>
  </body>
</html>
"""

soup = BeautifulSoup(html_doc, "html.parser")

# 1. 透過 ID 與 Class 尋找標題
title_node = soup.select_one("#main .title")
print(f"標題文字: {title_node.text}")

# 2. 獲取所有新聞連結
links = soup.select(".news-list > li > a")
for link in links:
    href = link.get("href") # 取得屬性值
    text = link.text        # 取得文字
    print(f"連結: {href}, 標題: {text}")
```

## 四、 lxml 中的 XPath 實戰

在 Python 中，通常會使用 `lxml.etree` 模組來解析 HTML 並套用 XPath 定位：

```python
from lxml import etree

html_content = """
<div id="menu">
  <span class="category">程式語言</span>
  <ul>
    <li><a href="/python">Python 基礎</a></li>
    <li><a href="/java">Java 基礎</a></li>
  </ul>
</div>
"""

# 解析 HTML 建立樹狀結構
tree = etree.HTML(html_content)

# 1. 精確定位類別名稱文字
category = tree.xpath("//div[@id='menu']/span/text()")
print(f"分類: {category[0]}")

# 2. 獲取特定文字標籤的 href 屬性 (向下搜尋)
python_link = tree.xpath("//a[text()='Python 基礎']/@href")
print(f"Python 連結: {python_link[0]}")

# 3. 獲取所有連結與文字
links = tree.xpath("//div[@id='menu']//li/a")
for a in links:
    # 在定位到的節點上，使用相對路徑 '.' 進行二次搜尋
    href = a.xpath("./@href")[0]
    text = a.xpath("./text()")[0]
    print(f"路徑: {href}, 內容: {text}")
```
