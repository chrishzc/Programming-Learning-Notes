# 5. 資料管道與 ETL 流程

> [!ABSTRACT]
> 本章介紹資料工程中的核心概念：資料管道 (Data Pipeline) 與 ETL 流程，並學習如何將提取、轉換、載入應用在網路爬蟲的資料清理實務中。

---

## 一、 什麼是資料管道 (Data Pipeline)？
資料管道是指資料從一個系統流向另一個系統的自動化過程。在網路爬蟲中，我們通常不會直接拿原始 HTML 去做分析，而是需要將資料經過提取、過濾、格式化，最後儲存為乾淨的結構化資料，這段連續流動的處理程序即為資料管道。

---

## 二、 ETL + A 核心流程
ETL 是資料倉庫 (Data Warehouse) 與大數據處理中最經典的架構模型：

```mermaid
graph LR
    E[Extract 提取] --> T[Transform 轉換]
    T --> L[Load 載入]
    L --> A[Application 應用]
```

### 1. E (Extract) - 提取
從各種異質資料來源中獲取原始資料。
- **結構化資料**：如關係型資料庫、CSV 表格。
- **半結構化資料**：如 JSON 檔案、XML 檔案、YAML 設定檔。
- **非結構化資料**：如原始 HTML 網頁原始碼、純文字檔、PDF 報告。
- *爬蟲實務*：利用 `requests` 請求網頁，並使用 `BeautifulSoup` 抓取特定區塊。

### 2. T (Transform) - 轉換
對提取出的原始資料進行清洗、重構與加工，這是確保資料品質的黃金步驟。
- **資料清洗**：填充缺失值（NaN）、刪除重複資料、剔除極端異常值。
- **格式規範化**：字串去除首尾空白（`.strip()`）、將日期文字轉為標準 `YYYY-MM-DD` 格式、數值轉型。
- **資料去識別化**：對敏感性個資（如電話、身份證字號）進行遮蔽或雜湊。

### 3. L (Load) - 載入
將轉換乾淨的資料寫入目標位置，以供後續使用。
- **本機載入**：儲存為 CSV、JSON 或 Excel 檔案。
- **資料庫載入**：寫入關聯式資料庫（MySQL、PostgreSQL）或非關聯式資料庫（MongoDB）。

### 4. A (Application) - 應用
ETL 的最終目的是將乾淨的資料轉化為商業價值。
- **BI 報表與視覺化**：使用 Tableau、PowerBI 或 Matplotlib 繪製統計圖表。
- **模型訓練**：將清洗好的特徵資料輸入機器學習模型中進行訓練與預測。

## 三、 實戰案例：簡易的 ETL 爬蟲管道

```python
import requests
from bs4 import BeautifulSoup
import pandas as pd

# Target: 提取並清洗範例網站的商品資訊
url = "https://example.com/products"

# === 1. Extract (提取) ===
try:
    response = requests.get(url, timeout=5)
    soup = BeautifulSoup(response.text, "html.parser")
    products = soup.select(".product-card")
except Exception as e:
    print(f"提取失敗: {e}")
    products = []

raw_data = []
for p in products:
    name = p.select_one(".name").text if p.select_one(".name") else None
    price = p.select_one(".price").text if p.select_one(".price") else None
    raw_data.append({"name": name, "price": price})

# 假設 raw_data = [{"name": "  iPhone 15  ", "price": "$29,900元 "}, {"name": "iPad", "price": None}]

# === 2. Transform (轉換) ===
df = pd.DataFrame(raw_data)

# 清除名字的首尾空白
df["name"] = df["name"].str.strip()

# 清洗價格：移除錢字號與中文，轉為整數數值，並將缺失值填補為 0
df["price"] = df["price"].str.replace(r"[^\d]", "", regex=True)
df["price"] = pd.to_numeric(df["price"]).fillna(0).astype(int)

# === 3. Load (載入) ===
df.to_csv("clean_products.csv", index=False, encoding="utf-8-sig")
print("ETL Data Pipeline 執行完成！")
```
