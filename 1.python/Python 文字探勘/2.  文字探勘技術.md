# 2. 文字探勘核心技術與挑戰

> [!ABSTRACT]
> 本章介紹文字探勘（Text Mining）與自然語言處理（NLP）的核心技術，包含分詞、詞性標註、命名實體辨識（NER）、依存分析等基礎技術；並說明深層語意分析方法（語意角色標註與語意依存分析），最後彙整在不同處理層次（字元、詞、句子、段落、文件及語料庫）中所面臨的文字處理挑戰。

---

## 一、 基礎文本處理技術 (分詞、詞性標註與依存)

在文字探勘的預處理階段，以下幾項基礎技術是不可或缺的基石：

| Text Mining 項目 | 說明 |
| :--- | :--- |
| **分詞<br>(Word Segmentation, WS)** | 切分文章中的詞序列。分詞是資訊檢索、文本挖掘、語意分析等自然語言處理應用的基礎。 |
| **詞性標註<br>(Part-of-speech Tagging, POS)** | 辨識句子中每個詞的詞性（例如：名詞、動詞、形容詞等）。<br>常見的詞性標記簡稱：<br>- `v`：動詞<br>- `n`：名詞<br>- `c`：連詞<br>- `d`：副詞<br>- `wp`：標點符號。 |
| **命名實體辨識<br>(Named Entity Recognition, NER)** | 辨識句子中特定類別的實體，如人名、地名、機構名等。 |
| **依存分析<br>(Dependency Parsing, DP)** | 通過分析語言成分之間的依存關係來解析並說明其句法結構，識別句子中各個詞項之間的關聯與結構關係。 |

---

## 二、 語意分析技術

為了深入理解文本的真實含義，必須進一步進行語意分析：

| Text Mining 項目 | 說明 |
| :--- | :--- |
| **語意角色標註<br>(Semantic Role Labeling, SRL)** | 是一種淺層的語意分析技術。主要目的在於標註句子中的某些短語作為給定謂詞的論元（即語意角色），例如：施事者、受事者、時間和地點等。此技術對問答系統、資訊抽取和機器翻譯等高階應用有極大的推動作用。 |
| **語意依存分析<br>(Semantic Dependency Parsing, SDP)** | 分析句子中各個語言單位（詞彙）之間的語意關聯，並將此關聯以依存結構呈現。使用語意依存來刻畫句子語意的優點在於**不需要抽象詞彙本身**，而是通過詞彙所承受的語意框架來描述該詞彙，而論元的數目相對於詞彙量來說總是少很多的。其最終目標是跨越句子表層句法結構的束縛，直接獲取深層的語意資訊。 |

## 三、 文本處理的挑戰

在處理自然語言時，不同層級的文本架構皆面臨不同的技術挑戰：

| 處理層次 | 面臨的挑戰 |
| :--- | :--- |
| **字元 (Character)** | - **編碼問題**：不同系統間的編碼衝突（ASCII, Big5, Latin-1, UTF-8, UTF-16 等）。<br>- **字元清理**：大小寫轉換、標點符號過濾、全形與半形字元轉換、重音符號處理以及數字的識別。 |
| **詞 (Word) 與 詞素 (Morpheme)** | - **分詞**：中文等無明顯空格邊界的語言分詞難度高。<br>- **詞性標註**：多義詞的詞性判定。<br>- **同義詞辨識**：相同意思的不同詞彙對應。<br>- **詞還原 (Lemmatization)**：將變形詞還原為原型（如 `words` ➔ `word`）。<br>- **簡寫處理**：省略詞與縮寫的識別。 |
| **多詞與句子<br>(Multiword and sentence)** | - **短語識別**：多個詞彙組成的固定片語。<br>- **句法分析 (Parse)**：解析句子結構。<br>- **句子邊界判定**：英文等語言的句子邊界（如句點）較易判斷，但仍有例外；非英文語言的句子邊界識別通常是一項挑戰。 |
| **多句與段落<br>(Multisentence and paragraph)** | - **意圖理解**：了解整個段落或文本的真實意圖。<br>- **摘要演算法**：需要演算法去自動識別哪些句子對於表達文本的核心意思是最重要的。 |
| **文件 (Document)** | - **背景知識依賴**：理解一篇文章往往需要超過實際文本本身所能提供的知識（常識、上下文），這使得對整份文件進行摘要與理解成為一項挑戰。 |
| **多文件與語料庫<br>(Multidocument and corpus)** | - **巨量資料分析**：如何在龐大的文本集合中進行高效率分類。<br>- **興趣點偵測**：快速發現使用者感興趣或熱門的知識點。<br>- **多文件摘要**：跨越多個文件進行整合摘要，甚至基於語料庫自動產生新文本。 |
