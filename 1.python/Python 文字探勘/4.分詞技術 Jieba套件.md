# 5. 中文分詞技術與 Jieba 套件實務

> [!ABSTRACT]
> 本章介紹中文自然語言處理（NLP）中的核心技術——分詞（Word Segmentation），解析中研院 CKIP 系統與主流開源分詞工具 Jieba 的運作原理，並提供分詞模式、自定義辭典與關鍵字提取的 Python 實作指南。

---

## 一、 中文分詞的挑戰與 CKIP Lab

英文等印歐語系語言在書寫時，單字之間天然以空格（Space）區隔，因此分詞相對簡單。然而中文為連續書寫，字與字之間沒有空格，這給電腦在進行文本分析時帶來了極大挑戰。

例如：「**結合成分子**」可以切分為「結合/成分/子」或「結合/成/分子」，需要依賴上下文或語意才能精確判斷。

為了解決中文語意解析的問題，台灣中央研究院創立了 **[CKIP Lab (中文詞知識庫小組)](https://ckip.iis.sinica.edu.tw/)**，其核心功能包括：

- **斷詞系統 (Word Segmentation)**：將相黏的字串，依據語境正確切分為獨立的詞彙單位。
- **實體辨識 (Named Entity Recognition, NER)**：自動找出文本中的「專有名詞」（如人名、地名、機構、時間等）並進行分類。
- **剖析系統 (Parsing System)**：拆解整句話的語法結構，建立「句型結構樹」，分析主詞子句、動詞片語以及形容詞修飾關係。
- **其他高階功能（β 測試中）**：語義辨識、指代消解、關係抽取等。

---

## 二、 開源中文分詞利器：Jieba (結巴)

中研院 CKIP 系統準確度極高，但因運作負載大、早期授權較為嚴格；在開源社群中，最受歡迎的 Python 中文分詞套件是 **Jieba (結巴分詞)**。

### Jieba 的核心特點
1. **多種分詞模式**：提供精確模式、全模式、搜尋引擎模式以符合不同分析場景。
2. **支援繁體分詞**：內建繁體中文辭典，亦可透過載入大辭典提升繁體字切分準確度。
3. **支援自定義辭典**：可動態新增專業術語或流行詞彙，防止專有名詞被切碎。

---

## 三、 Jieba 三大分詞模式實作

Jieba 提供了以下三種主要的分詞切分方式：

| 分詞模式 | 原理與特性 | 適用場景 |
| :--- | :--- | :--- |
| **精確模式** | 試圖將句子最精確地切開，不產生冗餘詞彙 | 文本分析、情感分析、機器學習特徵提取 |
| **全模式** | 掃描句子中所有可以成詞的詞語，速度快但有歧義 | 快速詞頻統計、詞雲圖繪製前置處理 |
| **搜尋引擎模式** | 在精確模式基礎上，對長詞再次切分，提高召回率 | 搜尋引擎索引建置、全文檢索系統 |

### 💻 分詞程式碼範例

```python
import jieba

text = "電腦視覺與自然語言處理是人工智慧的重要分支"

# 1. 精確模式 (預設 cut_all=False)
seg_list_exact = jieba.cut(text, cut_all=False)
print("精確模式:", "/ ".join(seg_list_exact))

# 2. 全模式 (cut_all=True)
seg_list_all = jieba.cut(text, cut_all=True)
print("全模式:  ", "/ ".join(seg_list_all))

# 3. 搜尋引擎模式
seg_list_search = jieba.cut_for_search(text)
print("搜尋引擎:", "/ ".join(seg_list_search))
```

> [!NOTE]
> `jieba.cut` 回傳的是一個 Python 生成器（Generator），因此在輸出時通常會使用 `list()` 轉換，或以 `join()` 連接。

## 四、 進階應用：自定義辭典與動態調權

當處理特定領域的文本（如醫療、金融、ACG 或流行語）時，預設辭典可能無法辨識專有名詞。例如將「**吉伊卡哇**」切成「吉伊/卡/哇」。此時需要自定義辭典。

### 1. 載入外部自定義辭典檔
建立一個文字檔 `dict.txt`，格式如下（每行一個詞，空格分隔：`詞項 詞頻 詞性`，詞頻與詞性可省略）：
```text
吉伊卡哇 10 n
烏薩奇 10 n
```
在 Python 中載入該辭典：
```python
jieba.load_userdict("dict.txt")
```

### 2. 動態調整辭典 (程式內即時修正)

```python
import jieba

# 預設切分
print("調整前:", "/ ".join(jieba.cut("我很喜歡小八貓這個角色")))

# 動態新增詞彙
jieba.add_word("小八貓")
print("新增詞後:", "/ ".join(jieba.cut("我很喜歡小八貓這個角色")))

# 動態調整詞頻，防止特定片語被拆開
jieba.suggest_freq("小八貓", tune=True)

# 刪除特定詞彙
jieba.del_word("這個")
print("刪除詞後:", "/ ".join(jieba.cut("我很喜歡小八貓這個角色")))
```

## 五、 關鍵字提取 (Keyword Extraction)

Jieba 內建了基於 TF-IDF 演算法的關鍵字提取功能，能快速從一長段文章中抓取出最重要的代表詞。

```python
import jieba.analyse

text = """
自然語言處理是人工智慧和語言學領域中的重要方向。
它研究能實現人與計算機之間用自然語言進行有效通信的各種理論和方法。
隨著深度學習的發展，自然語言處理在翻譯、問答系統和文本分析中取得了巨大突破。
"""

# 提取前 3 個權重最高的關鍵字
keywords = jieba.analyse.extract_tags(text, topK=3, withWeight=True)

print("關鍵字提取結果:")
for word, weight in keywords:
    print(f"詞彙: {word:<10} | 權重 (TF-IDF): {weight:.4f}")
```

> [!TIP]
> **移除停用字 (Stopwords)**
> 在進行分詞或關鍵字提取時，建議額外載入停用字表（如「的」、「了」、「在」等無意義助詞），可以使用 `jieba.analyse.set_stop_words(file_path)` 排除這些干擾詞，讓分析結果更精準。

## 六、 詞性標註 (Part-of-Speech Tagging)

除了切分詞彙外，Jieba 還提供了標註詞性的功能（例如判斷某個詞是動詞、名詞還是形容詞），這對於更深度的文本分析非常有用。

```python
import jieba.posseg as pseg

text = "中研院開發了先進的中文剖析系統"

# 進行詞性標註切分
words = pseg.cut(text)

print("詞性標註結果:")
for word, flag in words:
    # word 為詞彙，flag 為詞性標記（如 n 代表名詞、v 代表動詞）
    print(f"詞彙: {word:<6} | 詞性標記: {flag}")
```

> [!NOTE]
> **常用詞性標記簡表**
> - **n**：名詞 (Noun)
> - **v**：動詞 (Verb)
> - **a**：形容詞 (Adjective)
> - **d**：副詞 (Adverb)
> - **eng**：英文單字 (English)
