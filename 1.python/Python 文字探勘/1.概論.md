# 1. Python 文字探勘概論

> [!ABSTRACT]
> 本章介紹文字探勘（Text Mining）與自然語言處理（NLP）的基礎概念，探討如何將非結構化的文本資料轉化為結構化資訊；並詳細說明資訊檢索（IR）系統的運作方式、NLP 的定義、其與文字探勘的區別，以及自然語言處理的主要研究範疇。

---

## 一、 文本挖掘 (Text Mining)

一篇文章（或者書籍、網頁）通常是**非結構化資料**，必須經過文本挖掘的過程，從中抽取出關鍵特徵，以得到便於分析的**結構化資料**。

---

## 二、 資訊檢索 (Information Retrieval, IR)

資訊檢索是指從大規模資訊資源集合中，獲得與使用者資訊需求相關的資訊資源的活動。搜尋過程可以基於全文檢索，亦可基於內容的索引。

- **資訊超載管理**：自動資訊檢索系統可用於減少所謂的「資訊超載」。許多大學和公共圖書館皆使用 IR 系統提供圖書、期刊和檔案的檢索服務。
- **Web 搜尋引擎**：這是目前最常見且廣為人知的 IR 應用程式。
- **索引與搜尋機制**：資訊檢索主要針對非結構化資料進行索引與搜尋。使用者利用檢索系統輸入關鍵字，檢索系統在後台進行搜尋與匹配，最終找出最相關的結果。

## 三、 自然語言處理 (NLP) 介紹

自然語言處理（Natural Language Processing, NLP）是人工智慧和語言學領域的分支學科。
- **核心目標**：此領域旨在探討如何處理及運用自然語言。其中，自然語言認知是指讓電腦能夠「懂」人類的語言。
- **NLG 與 NLU**：
  - **自然語言生成 (NLG)**：系統負責把計算機的結構化數據轉化為自然語言文本。
  - **自然語言理解 (NLU)**：系統負責將自然語言轉化為計算機程序更易於處理的形式。

> [!NOTE]
> **文字探勘 (Text Mining) 與 自然語言處理 (NLP) 的差異**
> - **Text Mining**：偏向處理「文字本身」。其目的在於將非結構化的文本內容轉為結構化資訊，從中取得有用的關鍵特徵或趨勢。
> - **NLP**：會深入考慮「文字組成的語法與語意結構」，進而理解文字背後所代表的真實意義。

### 自然語言處理的主要研究範疇
自然語言處理技術廣泛應用於以下領域：
- 文本朗讀（Text to speech）/ 語音合成（Speech synthesis）
- 語音識別（Speech recognition）
- 中文自動分詞（Chinese word segmentation）
- 詞性標註（Part-of-speech tagging）
- 句法分析（Parsing）
- 自然語言生成（Natural language generation）
- 文本分類（Text categorization）
- 信息檢索（Information retrieval）
- 信息抽取（Information extraction）
- 文字校對（Text-proofing）
- 問答系統（Question answering）
- 機器翻譯（Machine translation）

## 四、 Python NLP 核心套件與環境

在進行中文與英文文字探勘實務時，Python 擁有一套完整且強大的開源生態系。以下是進行文字探勘實驗室環境（NLP Lab）建置時所需的五大核心套件：

### 1. 核心套件清單

| 套件名稱 | 核心用途 | 安裝指令 |
| :--- | :--- | :--- |
| **`jieba`** | 中文斷詞、自定義詞典與詞性標註 | `pip install jieba` |
| **`scikit-learn`** | 機器學習演算法、詞袋模型 (BoW) 與 TF-IDF 特徵提取 | `pip install scikit-learn` |
| **`gensim`** | 詞向量 (Word2Vec) 模型訓練與語意相似度計算 | `pip install gensim` |
| **`nltk`** | 英文自然語言處理基礎工具箱（詞幹提取、語料庫） | `pip install nltk` |
| **`spacy`** | 工業級快速 NLP 處理、命名實體識別 (NER) 與句法分析 | `pip install spacy` |

### 2. 環境驗證程式碼

您可以執行以下 Python 程式碼，確保所有 NLP 開發環境套件均已正確安裝：

```python
import jieba
import sklearn
import gensim
import nltk
import spacy

print("Jieba 版本:", jieba.__version__)
print("Scikit-learn 版本:", sklearn.__version__)
print("Gensim 版本:", gensim.__version__)
print("NLTK 版本:", nltk.__version__)
print("SpaCy 版本:", spacy.__version__)
print("\n🎉 所有 NLP 核心開發套件載入成功！環境建置完成。")
```
