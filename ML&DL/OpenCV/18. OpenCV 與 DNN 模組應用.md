# 18. OpenCV 與 DNN 模組應用

> [!ABSTRACT]
> 本章介紹 OpenCV 中的 `dnn` (Deep Neural Network) 模組。該模組使開發者能夠在不安裝 TensorFlow、PyTorch 等龐大深度學習框架的情況下，直接載入已訓練好的深度學習模型進行高效率的前向推理。本章將介紹 Blob 圖像預處理、模型讀取，並提供載入 Caffe/SSD 模型進行即時人臉偵測的完整實戰程式碼。

---

## 一、 cv2.dnn 模組簡介

OpenCV 的 `dnn` 模組是一個專門用來執行**深度學習推理 (Inference / Forward Pass)** 的輕量級工具。

- **核心特色**：
  - **僅支援推理**：不支援模型的訓練（Training），只能載入預訓練模型並執行前向傳播獲取預測結果。
  - **高度跨框架相容**：支援直接讀取 TensorFlow (`.pb`)、Caffe (`.caffemodel`)、Darknet/YOLO (`.weights`) 以及通用交換格式 ONNX (`.onnx`) 格式的模型。
  - **效能優異**：經過 Intel CPU/GPU 特殊優化，在嵌入式裝置或無 GPU 的伺服器上執行速度極快。

---

## 二、 影像轉換為 Blob (blobFromImage)

深度學習模型無法直接接收原始的 OpenCV BGR 影像矩陣。在輸入模型之前，必須先對影像進行縮放、減去均值、通道調換與歸一化等預處理，這在 OpenCV 中被稱為將影像轉換成 **Blob**。

```python
blob = cv2.dnn.blobFromImage(image, scalefactor=1.0, size=(width, height), mean=(R, G, B), swapRB=True, crop=False)
```
- **`image`**：輸入的原始彩色影像。
- **`scalefactor`**：像素值的縮放因子（例如：將 `0~255` 的像素歸一化到 `0~1`，則設為 `1/255.0`）。
- **`size`**：模型要求輸入的特定影像尺寸（如 $300 \times 300$ 或 $224 \times 224$）。
- **`mean`**：要減去的通道均值（用於數據去中心化，例如 ImageNet 模型常用的 `(104.0, 177.0, 123.0)`）。
- **`swapRB`**：是否調換紅藍通道（因為 OpenCV 讀入是 BGR，而多數模型是 RGB，此參數常設為 `True`）。
- **`crop`**：是否在調整尺寸時進行裁剪。

## 三、 載入模型與前向傳播 (readNet & forward)

### 1. 載入模型網路
OpenCV 提供了多個特定的讀取函數，或者可以使用統一的通用加載函數：
```python
# 通用加載方式
net = cv2.dnn.readNet(model_file, config_file)

# 例如 Caffe 模型：
net = cv2.dnn.readNetFromCaffe(prototxt_path, caffemodel_path)

# 例如 ONNX 模型：
net = cv2.dnn.readNetFromONNX(onnx_path)
```

### 2. 輸入與推理
將 Blob 輸入網路，並執行前向傳播：
```python
net.setInput(blob)
detections = net.forward() # 執行前向傳播，獲取模型預測結果
```

## 四、 實戰演練：載入 Caffe/SSD 模型進行人臉偵測

以下程式碼示範如何載入 OpenCV 內建的輕量級 ResNet-10 SSD 人臉偵測模型（Caffe 格式），並在視訊畫面中標記出偵測到的人臉：

```python
import cv2
import numpy as np

# 1. 定義 Caffe 模型權重與配置檔路徑
model_path = "res10_300x300_ssd_iter_140000.caffemodel"
config_path = "deploy.prototxt"

# 2. 載入 Caffe 網路
net = cv2.dnn.readNetFromCaffe(config_path, model_path)

# 3. 開啟攝影機
cap = cv2.VideoCapture(0)

while True:
    ret, frame = cap.read()
    if not ret:
        break
        
    h, w = frame.shape[:2]
    
    # 4. 將影像轉換為模型要求的 300x300 Blob 數據
    blob = cv2.dnn.blobFromImage(frame, scalefactor=1.0, size=(300, 300), 
                                 mean=(104.0, 177.0, 123.0), swapRB=False)
                                 
    # 5. 將 Blob 輸入網路並執行前向推理
    net.setInput(blob)
    detections = net.forward()
    
    # 6. 解析偵測結果
    # detections 的 shape 為 (1, 1, 偵測到的人臉數, 7)
    for i in range(detections.shape[2]):
        confidence = detections[0, 0, i, 2] # 取得信賴度 (機率值)
        
        # 信心值大於 0.5 才判定為人臉
        if confidence > 0.5:
            # 取得人臉邊框相對座標，並還原至原始影像寬高
            box = detections[0, 0, i, 3:7] * np.array([w, h, w, h])
            x_start, y_start, x_end, y_end = box.astype("int")
            
            # 繪製紅框
            cv2.rectangle(frame, (x_start, y_start), (x_end, y_end), (0, 0, 255), 2)
            cv2.putText(frame, f"Face: {confidence*100:.1f}%", (x_start, y_start - 10),
                        cv2.FONT_HERSHEY_SIMPLEX, 0.5, (0, 0, 255), 1)

    # 7. 顯示人臉偵測結果
    cv2.imshow("OpenCV DNN Face Detector", frame)
    
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()
```
