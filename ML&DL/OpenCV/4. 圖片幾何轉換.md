# OpenCV 影像幾何轉換

> [!ABSTRACT]
> 本篇筆記介紹如何對影像進行空間幾何變換，包含影像縮放（Resize）、插值演算法選擇、翻轉（Flip）、平移（Translation）與旋轉（Rotation），並說明仿射變換矩陣（Affine Matrix）的原理與實作。

---

## 1. 影像插值演算法 (Interpolation)

當影像放大或縮小時，像素點的分佈會改變，需要透過插值演算法來「補色」或「重採樣」，以維持影像的視覺品質。

### 1.1 常用插值模式對照表
在 `cv2.resize` 中，可透過 `interpolation` 參數指定插值方法：

| 插值標記 (Flags) | 名稱 | 特性與說明 | 適用情境 |
| :--- | :--- | :--- | :--- |
| `cv2.INTER_NEAREST` | 最近鄰插值 | 速度最快。直接取最近點的顏色，邊緣會產生明顯的**鋸齒與馬賽克效能**。 | 不推薦使用 (除非需要保留精確像素類別) |
| `cv2.INTER_LINEAR` | 雙線性插值 | **預設值**。以鄰近 4 點的距離作為權重進行計算，速度與品質平衡良好。 | **放大**影像常用 |
| `cv2.INTER_AREA` | 區域插值 | 使用像素區域關係重採樣。縮小影像時能有效避免摩爾紋（Moiré pattern）。 | **縮小**影像之首選 |
| `cv2.INTER_CUBIC` | 雙三次插值 | 參考 4x4 鄰域像素，使用三次樣條函數計算。速度較慢，但能有效避免鋸齒。 | **放大**影像（效果比線性更好） |
| `cv2.INTER_LANCZOS4` | Lanczos 插值 | 參考 8x8 鄰域像素，以 Lanczos 窗函數進行計算。運算量最大，但影像邊緣最平滑。 | 高畫質缩放需求 |

## 2. 影像縮放 (Scaling / Resize)

### 2.1 語法
```python
dst = cv2.resize(src, dsize[, fx[, fy[, interpolation]]])
```
> [!IMPORTANT]
> **維度順序差異**
> - NumPy 影像的形狀為 `(高度/Rows, 寬度/Cols)`。
> - `cv2.resize` 的目標尺寸 `dsize` 順序是 **`(寬度/Width, 高度/Height)`**。這點非常容易混淆！

### 2.2 程式範例
```python
import cv2

img = cv2.imread('image.jpg')
h, w = img.shape[:2]

# 1. 縮放到指定尺寸 (寬 600, 高 300)
resized_1 = cv2.resize(img, (600, 300), interpolation=cv2.INTER_LINEAR)

# 2. 等比例調整 (寬高均縮小為 0.5 倍)
# 當使用比例 fx, fy 時，dsize 必須帶入 (0, 0) 或 None
resized_2 = cv2.resize(img, (0, 0), fx=0.5, fy=0.5, interpolation=cv2.INTER_AREA)
```

## 3. 影像翻轉 (Flip)

影像翻轉是指將影像繞著對稱軸作鏡像映射。

### 3.1 語法
```python
dst = cv2.flip(src, flipCode)
```

### 3.2 翻轉代碼說明
- **`flipCode = 0`**：繞著 **X 軸**（水平軸）對稱翻轉，即**上下翻轉**（Vertical Flip）。
- **`flipCode > 0` (通常設為 1)**：繞著 **Y 軸**（垂直軸）對稱翻轉，即**左右翻轉**（Horizontal Flip，鏡像）。
- **`flipCode < 0` (通常設為 -1)**：繞著 **X 軸與 Y 軸**同時對稱翻轉，即**上下左右對調 / 旋轉 180 度**。

## 4. 影像平移 (Translation)

影像平移是仿射變換（Affine Transformation）的一種特例。需構建一個 $2 \times 3$ 的變換矩陣 $M$：

$$M = \begin{bmatrix} 1 & 0 & t_x \\ 0 & 1 & t_y \end{bmatrix}$$

*其中 $t_x$ 為水平平移距離（右移為正，左移為負），$t_y$ 為垂直平移距離（下移為正，上移為負）。*

### 4.1 程式範例
```python
import cv2
import numpy as np

img = cv2.imread('image.jpg')
h, w = img.shape[:2]

# 設定平移量：向右平移 100 像素，向下平移 50 像素
tx, ty = 100, 50

# 建立 2x3 的浮點型平移矩陣 M
M = np.float32([[1, 0, tx], [0, 1, ty]])

# 執行平移變換
# dsize 指定輸出影像大小 (width, height)
shifted_img = cv2.warpAffine(img, M, (w, h))
```

## 5. 影像旋轉 (Rotation)

旋轉影像需要先建立包含旋轉中心、角度與縮放比例的 $2 \times 3$ 旋轉矩陣 $M$，再以仿射變換套用。

### 5.1 任意角度旋轉 (搭配變換矩陣)
#### 語法：
```python
M = cv2.getRotationMatrix2D(center, angle, scale)
```
- `center`：旋轉中心點座標 `(x, y)`。
- `angle`：旋轉角度。**正值代表逆時針旋轉，負值代表順時針旋轉**。
- `scale`：影像縮放因子（例如 `1.0` 表示原大小，`0.5` 表示縮小一半）。

#### 程式範例：
```python
# 以影像中心為中心，逆時針旋轉 45 度，並縮小為 0.8 倍
center_pt = (w // 2, h // 2)
M = cv2.getRotationMatrix2D(center_pt, 45, 0.8)

# 套用仿射變換進行旋轉
rotated_img = cv2.warpAffine(img, M, (w, h))
```

### 5.2 快速直角旋轉 (`cv2.rotate`)
若只需要進行 $90^{\circ}$、$-90^{\circ}$ 或 $180^{\circ}$ 旋轉，使用 `cv2.rotate` 比 `warpAffine` 更快，且不會產生邊緣黑邊。

#### 旋轉角度常數：
- `cv2.ROTATE_90_CLOCKWISE`：順時針旋轉 90 度。
- `cv2.ROTATE_180`：旋轉 180 度。
- `cv2.ROTATE_90_COUNTERCLOCKWISE`：逆時針旋轉 90 度。

```python
# 順時針旋轉 90 度
rotated_90 = cv2.rotate(img, cv2.ROTATE_90_CLOCKWISE)
```
