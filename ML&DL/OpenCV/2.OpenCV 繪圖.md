## 1. 繪製基礎圖形

OpenCV 提供了多個繪圖函式，可用於在影像上繪製直線、矩形、圓形、橢圓及多邊形。

### 1.1 常用繪圖 API
- **畫線**：`cv2.line(img, pt1, pt2, color, thickness)`
- **畫箭頭**：`cv2.arrowedLine(img, pt1, pt2, color, thickness)`
- **畫矩形**：`cv2.rectangle(img, pt1, pt2, color, thickness)`
  - *提示：若 `thickness` 設為負值（或 `cv2.FILLED`），則會繪製「實心」矩形。*
- **畫圓**：`cv2.circle(img, center, radius, color, thickness)`
- **畫橢圓**：`cv2.ellipse(img, center, axes, angle, startAngle, endAngle, color, thickness)`
- **畫多邊形 / 折線**：`cv2.polylines(img, pts, isClosed, color, thickness)`

### 1.2 繪圖完整範例
以下範例建立一個 512x512 的黑色畫布，並在上面繪製上述所有圖形：

```python
import cv2
import numpy as np

# 建立一個 512x512 的黑色三通道畫布 (像素值均為 0)
canvas = np.zeros((512, 512, 3), dtype=np.uint8)

# 1. 畫線 (藍色 BGR: 255, 0, 0，線寬 3 像素)
cv2.line(canvas, (10, 10), (500, 500), (255, 0, 0), 3)

# 2. 畫箭頭 (綠色 BGR: 0, 255, 0，線寬 3 像素)
cv2.arrowedLine(canvas, (10, 500), (250, 260), (0, 255, 0), 3)

# 3. 畫矩形 (紅色 BGR: 0, 0, 255)
# 空心矩形 (線寬 2 像素)
cv2.rectangle(canvas, (50, 50), (150, 150), (0, 0, 255), 2)
# 實心矩形 (線寬設為 -1)
cv2.rectangle(canvas, (350, 50), (450, 150), (0, 0, 255), -1)

# 4. 畫圓形 (黃色 BGR: 0, 255, 255，半徑 60)
cv2.circle(canvas, (256, 256), 60, (0, 255, 255), 2)

# 5. 畫橢圓 (白色 BGR: 255, 255, 255)
# 參數：畫布, 中心點, (長軸半徑, 短軸半徑), 旋轉角度, 起始角度, 結束角度, 顏色, 線寬
# 若結束角度不足 360 度則會變為扇形形狀
cv2.ellipse(canvas, (256, 400), (100, 50), 30, 0, 360, (255, 255, 255), 2)

# 6. 畫多邊形 (粉色 BGR: 255, 0, 255)
# 定義頂點座標 (必須是 np.int32 型態)
pts = np.array([[50, 450], [150, 420], [100, 480]], np.int32)
# 需要將 points 重新 shape 成 (點數, 1, 2)
pts = pts.reshape((-1, 1, 2))
cv2.polylines(canvas, [pts], isClosed=True, color=(255, 0, 255), thickness=3)

# 顯示畫布
cv2.imshow('Canvas Drawing', canvas)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## 2. 影像算術運算與疊加

當對兩張影像進行色彩數值相加時，有兩種主要方法：

### 2.1 運算元直接相加 (`img1 + img2`)
- **計算方式**：使用 NumPy 陣列的相加。若兩像素相加的值超過 `255`，會對 `256` 取餘數（即 `(value1 + value2) % 256`）。
- **現象**：相加結果可能會進位，導致像素亮度產生強烈突變（例如 `150 + 120 = 270`，取餘數後為 `14`，反而變得很暗）。

### 2.2 OpenCV 飽和相加 (`cv2.add`)
- **計算方式**：進行飽和運算。若兩像素相加的值超過 `255`，最大值將鎖定在 `255`。
- **現象**：過曝的部分會呈現純白色，防止畫面爆掉失真。
```python
# 飽和運算相加
result_add = cv2.add(img1, img2)
```

### 2.3 影像權重疊加 (`cv2.addWeighted`)
常用於將兩張圖片以不同透明度進行重疊混合。
- **混合公式**：
  $$\text{dst} = \text{src1} \times \alpha + \text{src2} \times \beta + \gamma$$
  *其中 $\alpha$ 和 $\beta$ 分別為兩張圖片的權重（透明度），$\gamma$ 為整體亮度調整常數（偏置值）。*

- **程式語法**：
  ```python
  dst = cv2.addWeighted(src1, alpha, src2, beta, gamma)
  ```
  > [!NOTE]
  > 進行疊加的 `src1` 與 `src2` 影像，其尺寸（高度、寬度）與通道數必須完全相同。

## 3. 繪製文字 (`cv2.putText`)

語法：
```python
cv2.putText(img, text, org, fontFace, fontScale, color, thickness, lineType)
```
- `org`：文字的左下角起始座標 `(x, y)`。
- `fontScale`：字型大小比例。

### 3.1 字型選擇 (fontFace)
OpenCV 內建 8 種字型樣式：

| 字型名稱 (Python API) | 常數代碼 | 說明 |
| :--- | :---: | :--- |
| `cv2.FONT_HERSHEY_SIMPLEX` | `0` | 單線無襯線字型 (預設) |
| `cv2.FONT_HERSHEY_PLAIN` | `1` | 小號單線無襯線字型 |
| `cv2.FONT_HERSHEY_DUPLEX` | `2` | 雙線無襯線字型 |
| `cv2.FONT_HERSHEY_COMPLEX` | `3` | 雙線有襯線字型 (經典) |
| `cv2.FONT_HERSHEY_TRIPLEX` | `4` | 三線有襯線字型 |
| `cv2.FONT_HERSHEY_COMPLEX_SMALL` | `5` | 小號有襯線字型 |
| `cv2.FONT_HERSHEY_SCRIPT_SIMPLEX` | `6` | 手寫草書體 |
| `cv2.FONT_HERSHEY_SCRIPT_COMPLEX` | `7` | 複雜手寫草書體 |

### 3.2 線條類型 (lineType)
- `cv2.LINE_4` (4) 或 `cv2.LINE_8` (8)：標準的 4 鄰域或 8 鄰域繪製線。
- `cv2.LINE_AA` (16)：**反鋸齒線條（Anti-aliased line）**。強烈建議在繪製文字時使用，能使文字邊緣呈現平滑、去除顆粒感，效果非常 premium。

範例：
```python
# 在畫布上書寫反鋸齒文字
cv2.putText(canvas, "OpenCV Note", (50, 100), cv2.FONT_HERSHEY_SIMPLEX, 1.5, (255, 255, 255), 2, cv2.LINE_AA)
```

## 4. 滑鼠互動事件

您可以透過註冊滑鼠回呼函式來擷取滑鼠動作（點擊、移動、拖曳等），進而實現豐富的互動效果。

### 4.1 常用的滑鼠事件常數 (Event)
| 事件名稱 (Python API) | 數值 | 說明 |
| :--- | :---: | :--- |
| `cv2.EVENT_MOUSEMOVE` | `0` | 滑鼠移動 |
| `cv2.EVENT_LBUTTONDOWN` | `1` | 左鍵點擊按下 |
| `cv2.EVENT_RBUTTONDOWN` | `2` | 右鍵點擊按下 |
| `cv2.EVENT_MBUTTONDOWN` | `3` | 中鍵（滾輪）點擊按下 |
| `cv2.EVENT_LBUTTONUP` | `4` | 左鍵放開 |
| `cv2.EVENT_RBUTTONUP` | `5` | 右鍵放開 |
| `cv2.EVENT_MBUTTONUP` | `6` | 中鍵放開 |
| `cv2.EVENT_LBUTTONDBLCLK` | `7` | 左鍵按兩下（雙擊） |
| `cv2.EVENT_RBUTTONDBLCLK` | `8` | 右鍵按兩下 |
| `cv2.EVENT_MBUTTONDBLCLK` | `9` | 中鍵按兩下 |

### 4.2 常用的拖曳與輔助鍵旗標 (Flags)
| 旗標名稱 (Python API) | 數值 | 說明 |
| :--- | :---: | :--- |
| `cv2.EVENT_FLAG_LBUTTON` | `1` | 按住左鍵拖曳中 |
| `cv2.EVENT_FLAG_RBUTTON` | `2` | 按住右鍵拖曳中 |
| `cv2.EVENT_FLAG_MBUTTON` | `4` | 按住中鍵拖曳中 |
| `cv2.EVENT_FLAG_CTRLKEY` | `8` | 同時按住 Ctrl 鍵 |
| `cv2.EVENT_FLAG_SHIFTKEY` | `16` | 同時按住 Shift 鍵 |
| `cv2.EVENT_FLAG_ALTKEY` | `32` | 同時按住 Alt 鍵 |

### 4.3 滑鼠互動程式碼範例
以下程式碼實作了在視窗中移動滑鼠會印出 BGR 色彩，且在點擊左鍵或右鍵時在畫面上繪製標記的完整互動：

```python
import cv2
import numpy as np

# 標準滑鼠互動回呼函式
def on_mouse(event, x, y, flags, param):
    # 傳入的 param 可用於與外部變數傳遞資料 (此處傳入包裝影像的字典)
    img = param['image']
    
    # 當滑鼠移動時，在終端機即時印出座標與該位置的像素色彩
    if event == cv2.EVENT_MOUSEMOVE:
        # 影像矩陣的維度是 [y, x]
        pixel = img[y, x]
        print(f"滑鼠位置: (x={x}, y={y}) | BGR: {pixel}", end='\r')
        
    # 當左鍵點擊時，在點擊位置畫一個綠色實心圓圈
    elif event == cv2.EVENT_LBUTTONDOWN:
        cv2.circle(img, (x, y), 8, (0, 255, 0), -1)
        print(f"\n[左鍵點擊] 於 (x={x}, y={y}) 畫下綠色標記")
        
    # 當右鍵點擊時，在點擊位置畫一個紅色空心圓圈
    elif event == cv2.EVENT_RBUTTONDOWN:
        cv2.circle(img, (x, y), 12, (0, 0, 255), 2)
        print(f"\n[右鍵點擊] 於 (x={x}, y={y}) 畫下紅色標記")

# 主程式
# 建立一個寬 600、高 400 的白色底圖畫布
canvas = np.ones((400, 600, 3), dtype=np.uint8) * 255

# 構建視窗並綁定滑鼠事件
cv2.namedWindow('Mouse Interaction')

# 透過 dict 包裝畫布，以便在 callback 中進行直接修改與更新
params = {'image': canvas}
cv2.setMouseCallback('Mouse Interaction', on_mouse, params)

print("滑鼠互動開始！請在視窗內進行操作：")
print("- 左鍵點擊：標記綠色實心圓")
print("- 右鍵點擊：標記紅色空心圓")
print("- 按下 ESC 鍵以關閉視窗")

while True:
    cv2.imshow('Mouse Interaction', canvas)
    # 使用 waitKey(10) 以保證 GUI 執行緒流暢運行並在按下 ESC (鍵值 27) 時跳出
    if cv2.waitKey(10) == 27:
        break

cv2.destroyAllWindows()
```

## 5. 滾動條控制項 (Trackbar)

滾動條是在 OpenCV 視窗中非常實用的動態控制項，常用於調參、閥值調整等。

### 5.1 API 說明
- **建立滾動條**：
  ```python
  cv2.createTrackbar(trackbarName, windowName, value, count, onChange)
  ```
  - `trackbarName`：滑桿顯示的名稱。
  - `windowName`：滑桿所依附的**視窗名稱**（該視窗必須先前已使用 `cv2.namedWindow()` 建立）。
  - `value`：初始預設值（整數）。
  - `count`：滑桿最大值（整數，最小值固定為 0）。
  - `onChange`：回呼函式，每當拉動滑桿時會自動呼叫，並將當前數值作為參數傳入該函式。

- **取得當前數值**：
  ```python
  value = cv2.getTrackbarPos(trackbarName, windowName)
  ```

### 5.2 實用範例：RGB 調色盤
以下範例建立一個黑色畫布，並透過三個滾動條（R、G、B）來即時改變畫布的背景色彩：

```python
import cv2
import numpy as np

# 回呼函式 (由於我們在主迴圈直接 polling 滾動條的值，此處定義一個 pass 空函式即可)
def nothing(x):
    pass

# 建立 300x512 的黑色畫布
palette_img = np.zeros((300, 512, 3), dtype=np.uint8)
cv2.namedWindow('RGB Palette')

# 建立滾動條來控制 R, G, B 各色彩分量
# 參數：拉桿名稱、依附視窗、初始值、最大值、回呼函式
cv2.createTrackbar('R', 'RGB Palette', 0, 255, nothing)
cv2.createTrackbar('G', 'RGB Palette', 0, 255, nothing)
cv2.createTrackbar('B', 'RGB Palette', 0, 255, nothing)

print("RGB 調色盤已啟動！")
print("調整 'R'、'G'、'B' 滑桿動態變換色彩。按下 ESC 鍵退出。")

while True:
    cv2.imshow('RGB Palette', palette_img)
    
    if cv2.waitKey(10) == 27:
        break
        
    # 即時獲取滾動條的值
    r = cv2.getTrackbarPos('R', 'RGB Palette')
    g = cv2.getTrackbarPos('G', 'RGB Palette')
    b = cv2.getTrackbarPos('B', 'RGB Palette')
    
    # 將畫布顏色更新 (注意 OpenCV 順序是 BGR)
    palette_img[:] = [b, g, r]

cv2.destroyAllWindows()
```
