## 1. 常見的色彩空間

色彩空間（Color Space）是用於表示顏色方法的抽像數學模型，也被稱為「色域」。

### 1.1 RGB 與 CMYK 色彩模式

| 比較項目 | RGB 色彩模式 💻 | CMYK 色彩模式 🖨️ |
| :--- | :--- | :--- |
| **主要媒介** | 螢幕、電視、手機、光線 | 印刷品、課本、紙張、油墨 |
| **混色原理** | **加色法（Additive）** ➕ <br>光線越加越亮 | **減色法（Subtractive）** ➖ <br>顏料越加越暗 |
| **顏色組成** | **R**ed (紅)、**G**reen (綠)、**B**lue (藍) | **C**yan (青)、**M**agenta (洋紅)、**Y**ellow (黃)、**K**ey (黑) |
| **完全混合** | 變成**純白色（255, 255, 255）** | 變成**深黑色（100, 100, 100, 100）** |
| **色彩範圍** | 色域較**廣**（能表現螢光、亮鮮色） | 色域較**窄**（許多螢幕亮色印不出來） |

---

### 1.2 灰階影像 (Grayscale / Gray)
灰階影像只有一個亮度通道，像素值介於 `0`（黑）到 `255`（白）。

#### 灰階轉換與整數最佳化
標準彩色轉灰階的亮度加權公式為：
$$Gray = R \times 0.299 + G \times 0.587 + B \times 0.114$$

在硬體底層或早期嵌入式裝置中，浮點數乘法運算速度較慢。為了最佳化效能，會將係數放大後進行整數運算：
1. **放大 1000 倍運算**：
   $$Gray = \frac{299 \times R + 587 \times G + 114 \times B}{1000}$$
2. **二進位位移最佳化（放大 256 倍，即 $2^8$）**：
   在電腦科學中，除以 256 可以用極快的「位元右移 8 位元（`>> 8`）」替代，能大幅節省 CPU 運算開銷：
   $$Gray = (76 \times R + 150 \times G + 30 \times B) \gg 8$$

### 1.3 HSV 色彩空間
HSV 是基於人類視覺直覺設計的色彩空間，特別適合用於**顏色辨識與追蹤**。
1. **H (Hue，色相 / 顏色種類)**：
   - 代表光譜上的顏色種類（如紅、橙、黃、綠、藍、紫）。
   - 在物理上是 $360^{\circ}$ 的圓環。
   - **OpenCV 特殊設定**：因為 8 位元影像（`uint8`）最大只能儲存到 255，放不下 360。因此 **OpenCV 將 H 角度除以 2，範圍限縮在 0 ~ 180**。
2. **S (Saturation，飽和度 / 鮮豔程度)**：
   - 代表顏色的純度或濃淡。範圍為 `0 ~ 255`。數值越低越接近灰色，越高越鮮豔。
3. **V (Value，明度 / 亮度)**：
   - 代表光線的明亮程度。範圍為 `0 ~ 255`。`0` 為純黑，`255` 為最亮。

#### 常見色彩在 OpenCV 中的大約 HSV 範圍：
- **紅色 (Red)**：$0 \sim 10$ 及 $170 \sim 180$（因為色環首尾相接）
- **黃色 (Yellow)**：$26 \sim 34$
- **綠色 (Green)**：$35 \sim 77$
- **藍色 (Blue)**：$100 \sim 124$

---

### 1.4 Lab 色彩空間
Lab 色彩空間是一種感知均勻的色彩模型，與設備無關。
- **L**：Lightness（亮度，範圍 `0 ~ 100`）。
- **a**：綠色到洋紅色（Green $\to$ Magenta）的色彩分量。
- **b**：藍色到黃色（Blue $\to$ Yellow）的色彩分量。
> [!NOTE]
> 在 Lab 空間中，亮度資訊（L）與色彩資訊（a, b）是完全分離的，這與 RGB 的三個通道同時編碼亮度與色彩不同。因此非常適合用於需要排除光影變化干擾的影像處理。

## 2. 色彩空間轉換 (`cv2.cvtColor`)

OpenCV 提供了 `cv2.cvtColor` 進行不同色彩空間的轉換。

### 2.1 語法
```python
dst = cv2.cvtColor(src, code)
```

### 2.2 常用轉換常數 (Flags)
| 轉換常數 | 說明 |
| :--- | :--- |
| `cv2.COLOR_BGR2GRAY` | 將彩色 BGR 影像轉換為單通道灰階影像 |
| `cv2.COLOR_BGR2HSV` | 將彩色 BGR 影像轉換為 HSV 色彩空間 |
| `cv2.COLOR_HSV2BGR` | 將 HSV 影像轉換回 BGR 彩色影像 |
| `cv2.COLOR_BGR2LAB` | 將彩色 BGR 影像轉換為 Lab 色彩空間 |
| `cv2.COLOR_GRAY2BGR` | 將單通道灰階複製為三通道（常用於疊加彩色圖形） |

## 3. 影像遮罩與位元運算 (Masking & Bitwise)

### 3.1 遮罩（Mask）的基本概念
遮罩就像是一張剪紙模板，用於指定影像處理的**感興趣區域（ROI）**。
- 遮罩是一張二值化影像：**白色 (255)** 代表保留/處理區域，**黑色 (0)** 代表屏蔽/忽略區域。
- **主要應用情境**：
  1. **去背與裁剪**：透過位元 AND 運算將白色遮罩區域以外的背景過濾為黑色。
  2. **局部修圖**：傳入遮罩參數（如 `mask=mask`），使模糊或美白等濾鏡僅作用於特定部位（如人臉）。
  3. **隱私遮蔽**：在車牌或人臉處在遮罩上標記為黑色，相乘後即可完成去識別化。
  4. **特徵統計**：限制如 `cv2.mean(img, mask=mask)` 等統計函式僅計算特定區域的像素平均值。

---

### 3.2 影像位元邏輯運算
| 運算類型 | 運算規則 | Python 符號 | NumPy 陣列語法 | OpenCV 影像函式 |
| :--- | :--- | :---: | :--- | :--- |
| **NOT (非)** | 黑白翻轉（0 $\leftrightarrow$ 255） | `~x` | `np.invert(x)` | `cv2.bitwise_not(img)` |
| **AND (與)** | 兩者皆為白（255），結果才是白 | `x & y` | `np.bitwise_and(x, y)` | `cv2.bitwise_and(img1, img2, mask)` |
| **OR (或)** | 只要有一者為白，結果即為白 | `x \| y` | `np.bitwise_or(x, y)` | `cv2.bitwise_or(img1, img2)` |
| **XOR (互斥)** | 兩者不相同時為白，相同時為黑 | `x ^ y` | `np.bitwise_xor(x, y)` | `cv2.bitwise_xor(img1, img2)` |

---

## 4. 實戰範例：HSV 色彩篩選與去背

以下程式碼示範如何使用 `cv2.inRange` 設定綠色的 HSV 閥值範圍來產生遮罩，並使用 `cv2.bitwise_and` 提取綠色影像區塊：

```python
import cv2
import numpy as np

# 1. 建立一個虛擬測試彩色影像 (方便直接執行)
# 建立一個 300x400、背景為黑色的三通道影像
img = np.zeros((300, 400, 3), dtype=np.uint8)
# 在影像上繪製三個不同顏色的實心圓圈
cv2.circle(img, (100, 100), 50, (0, 255, 0), -1)    # 綠色圓圈 (BGR: 0, 255, 0)
cv2.circle(img, (300, 100), 50, (0, 0, 255), -1)    # 紅色圓圈 (BGR: 0, 0, 255)
cv2.circle(img, (200, 220), 50, (0, 255, 255), -1)  # 黃色圓圈 (BGR: 0, 255, 255)

# 2. 將 BGR 彩色影像轉換至 HSV 色彩空間
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)

# 3. 定義欲篩選的顏色範圍 (以「綠色」為例)
# 綠色 HSV 的範圍大致為：H: 35~77, S: 43~255, V: 46~255
lower_green = np.array([35, 43, 46])
upper_green = np.array([77, 255, 255])

# 4. 利用 cv2.inRange 產生色彩遮罩
# 在閥值區間內的像素會被設為 255 (白色)，區間外的像素被設為 0 (黑色)
green_mask = cv2.inRange(hsv, lower_green, upper_green)

# 5. 透過位元 AND 運算，結合原圖與遮罩，將綠色圓圈提取出來
# 只有在遮罩值為白色 (255) 的區域，才會保留原圖像素的顏色，其餘皆變成黑色 (0)
extracted_result = cv2.bitwise_and(img, img, mask=green_mask)

# 顯示各個階段的成果
cv2.imshow('Original Image', img)
cv2.imshow('Green Mask', green_mask)
cv2.imshow('Extracted Result', extracted_result)

print("提取綠色物件成功！")
print("按下任意鍵關閉視窗。")

cv2.waitKey(0)
cv2.destroyAllWindows()
```
