## 一、 TensorFlow 核心架構

**TensorFlow** 最初由 Google Brain 團隊開發，原本是其內部的機器學習工具。2015 年 11 月，Google 宣佈將其開源並遵守 Apache 2.0 協議。目前它已成為全球最主流的深度學習框架之一，支援各式不同的深度學習演算法。

### 1. 什麼是張量 (Tensor) 與流 (Flow)？
- **張量 (Tensor)**：TensorFlow 中的基本資料結構。不論是輸入的特徵、模型權重還是輸出的預測值，在底層都是以多維陣列（即「張量」）的形式存在。
- **流 (Flow) 與計算圖**：計算圖（Computational Graphs）是一種描述數學運算的低階圖形結構（如下圖所示）。建立的資料張量（Tensor）會在計算圖的路徑上流動（Flow）並進行運算，最終輸出結果。這也是 **Tensor + Flow = TensorFlow** 的由來。

## 二、 常量 (tf.constant) 與 變量 (tf.Variable)

在 TensorFlow 中，資料的儲存主要分為「不可變的常數」與「可隨訓練更新的變數」：

| 特性 | tf.constant (常數) | tf.Variable (變數) |
| :--- | :--- | :--- |
| **可變性** | 不可變 (Immutable) | 可變 (Mutable) |
| **數值修改方式** | 無法修改 | 必須使用 `.assign()`, `.assign_add()` 等方法 |
| **梯度追蹤** | 預設不追蹤 (不參與模型訓練調整) | 預設自動追蹤 (會隨模型訓練自動更新) |
| **主要用途** | 儲存輸入資料、固定超參數、常數 (如 $\pi$) | 儲存模型權重 (Weights) 與偏差 (Biases) |
| **記憶體開銷** | 較小 | 較大 (需記錄狀態與梯度資訊) |

### 實務宣告範例
```python
import tensorflow as tf

# 1. 宣告常數 (tf.constant)
a = tf.constant(3.0, dtype=tf.float32)
b = tf.constant(5.0, dtype=tf.float32)
c = a + b
print("常數相加結果:", c.numpy())  # 輸出: 8.0

# 2. 宣告變數 (tf.Variable)
# 變數通常需要給予初始值，並可用於梯度計算
w = tf.Variable(2.0, name="weight")
b_var = tf.Variable(1.0, name="bias")

# 修改變數的值 (注意：不可直接使用 w = 4.0，這會將變數覆蓋為普通 Python 常數)
w.assign(4.0)       # 將 w 的值修改為 4.0
w.assign_add(1.0)   # 將 w 的值自增 1.0 (等同於 w += 1)
print("變數 w 修改後的數值:", w.numpy())  # 輸出: 5.0
```

## 三、 Eager Execution (即時執行) 與 Graph Mode (計算圖模式)

> [!TIP]
> **TensorFlow 2.x 的圖模式轉換**
> - **Eager Execution (即時執行/動態圖)**：TensorFlow 2.x 的預設模式。運算會像普通 Python 程式碼一樣立即執行並回傳結果，便於除錯與交互開發。
> - **Graph Mode (計算圖模式/靜態圖)**：將運算編譯成最佳化的靜態計算圖後執行，速度極快，且易於跨平台與行動端部署。

我們可以使用 **`@tf.function`** 裝飾器，輕鬆將一個動態執行的 Python 函數編譯為高效的靜態計算圖：
```python
# 使用 @tf.function 將計算編譯為靜態圖以加速運行
@tf.function
def compute_loss(x, y, w, b):
    pred = x * w + b
    loss = tf.reduce_mean(tf.square(y - pred)) # 均方誤差 (MSE)
    return loss
```

## 四、 高階 API：Keras 及其整合

- **Keras** 是一個開放原始碼的深度學習庫，最初基於 TensorFlow、Theano 等後端進行封裝。它採用純 Python 編寫，專注於提供高層神經網絡 API。
- **核心優勢**：Keras 已經將訓練模型的輸入層、隱藏層、輸出層等底層數學架構封裝完畢。開發者無需手動編寫矩陣相乘等底層代碼，只需拼接圖層並設定正確參數（如神經元個數、激活函數）即可。
- **無縫集成**：由於 Keras 易用性極佳且與 TensorFlow 完美互補，自 **TensorFlow 2.0** 版本起，官方直接將其納入核心函式庫中，並命名為 **`tf.keras`**。

### Keras 搭建神經網路範例
```python
import tensorflow as tf
from tensorflow.keras import layers, models

# 1. 建立一個 Sequential (順序) 模型
model = models.Sequential([
    # 第一層隱藏層：使用 Dense (全連接層)，內含 32 個神經元，使用 ReLU 激活函數
    # 當作為首層時，需透過 input_shape 指定輸入特徵的維度 (例如：10)
    layers.Dense(32, activation='relu', input_shape=(10,)),
    
    # 第二層隱藏層：16 個神經元，使用 ReLU 激活函數
    layers.Dense(16, activation='relu'),
    
    # 輸出層：1 個神經元，使用 Sigmoid 激活函數 (適用於二分類機率輸出)
    layers.Dense(1, activation='sigmoid')
])

# 2. 輸出模型結構與參數量 (Weights & Biases) 資訊
model.summary()
```
